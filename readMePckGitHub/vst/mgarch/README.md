    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-mgarch---r-package-for-mgarch-model-simulation-and-estimation" class="anchor" href="#mgarch---r-package-for-mgarch-model-simulation-and-estimation" aria-hidden="true"><span class="octicon octicon-link"></span></a>mgarch - R Package for MGARCH Model Simulation and Estimation</h1>

<p><em>mgarch</em> is an R package for simulating and estimating multivariate
GARCH models. Currently, only BEKK, GJR and DCC specifications are
implemented.</p>

<h2>
<a id="user-content-basic-usage" class="anchor" href="#basic-usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Basic Usage</h2>

<p>After you install the package, you can load the library:</p>

<pre><code>&gt; library(mgarch)
</code></pre>

<p>Probably, you are interested in the following functions:</p>

<ol class="task-list">
<li>
<code>mvBEKK.sim</code>: Simulates an MGARCH-BEKK process</li>
<li>
<code>mvBEKK.est</code>: Estimates an MGARCH-BEKK model</li>
<li>
<code>mvBEKK.diag</code>: Provides diagnostics for an estimated MGARCH-BEKK
model</li>
</ol>

<p>You can get help from the embedded documentation by issuing:</p>

<pre><code>&gt; ?mvBEKK.sim
&gt; ?mvBEKK.est
&gt; ?mvBEKK.diag
</code></pre>

<p>As an example, you can simulate an MGARCH-BEKK process as follows:</p>

<pre><code>&gt; sim &lt;- mvBEKK.sim(series.count=2, T=1200)
</code></pre>

<p>This will give you two vectors of 2500 observations along with many
other objects. Checkout the help for further details.</p>

<p>Now create a data frame with the given two vectors:</p>

<pre><code>&gt; eps &lt;- data.frame(s1=sim$eps[[1]], s2=sim$eps[[2]])
</code></pre>
