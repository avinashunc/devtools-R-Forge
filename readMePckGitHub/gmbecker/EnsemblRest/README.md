<h2>
<a id="user-content-ensemblrest-an-r-package-for-accessing-the-ensembl-rest-api" class="anchor" href="#ensemblrest-an-r-package-for-accessing-the-ensembl-rest-api" aria-hidden="true"><span class="octicon octicon-link"></span></a>EnsemblRest an R package for accessing the Ensembl REST API.</h2>

<p><a href="http://beta.rest.ensembl.org/">http://beta.rest.ensembl.org/</a></p>

<p>Still very much in development.</p>

<p>To install, clone this repository and type:</p>

<pre><code>R CMD build EnsemblRest
R CMD INSTALL EnsemblRest_1.5.2.tar.gz
</code></pre>

<p>To load the package into an R session, simply type:</p>

<pre><code>library( EnsemblRest )
</code></pre>

<p>Then, for documentation from inside R, type:</p>

<pre><code>?EnsemblRest
</code></pre>

<p>Or, you can view the vignette by typing:</p>

<pre><code>print( vignette( 'EnsemblRest' ) )
</code></pre>

<p><a href="https://github.com/acbb/EnsemblRest/blob/vignette/EnsemblRest.pdf?raw=true">The current vignette is also available here</a></p>
