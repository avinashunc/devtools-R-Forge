    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-trap" class="anchor" href="#trap" aria-hidden="true"><span class="octicon octicon-link"></span></a>tRap</h1>

<p><a href="https://travis-ci.org/matthuska/tRap"><img src="https://camo.githubusercontent.com/658e69deaeba20471202dddb45ac90fb6b2d7375/68747470733a2f2f7472617669732d63692e6f72672f6d6174746875736b612f745261702e706e673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/matthuska/tRap.png?branch=master" style="max-width:100%;"></a></p>

<p>tRap is an R package that can be used to calculate the predicted binding affinity of a transcription factor to a DNA or RNA sequence. It implements the model described in <a href="http://www.ncbi.nlm.nih.gov/pubmed/17098775">Roider et al 2007</a>.</p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>You will need the latest version of <code>devtools</code>. First install the release version:</p>

<div class="highlight highlight-R"><pre>install.packages(<span class="pl-s"><span class="pl-pds">"</span>devtools<span class="pl-pds">"</span></span>)</pre></div>

<p>then update it to the latest developement version:</p>

<div class="highlight highlight-R"><pre><span class="pl-e">devtools</span><span class="pl-k">::</span>install_github(<span class="pl-s"><span class="pl-pds">"</span>devtools<span class="pl-pds">"</span></span>)</pre></div>

<p>Finally you can install the latest development version of tRap from github with:</p>

<div class="highlight highlight-R"><pre><span class="pl-e">devtools</span><span class="pl-k">::</span>install_github(<span class="pl-s"><span class="pl-pds">"</span>matthuska/tRap<span class="pl-pds">"</span></span>)</pre></div>

<p>If you're in our group at the MPIMG then you can also install a package containing pre-processed TRANSFAC matrices. Sorry this is not available to people outside of our group because TRANSFAC is a commercial database.</p>

<div class="highlight highlight-R"><pre><span class="pl-e">devtools</span><span class="pl-k">::</span>install_local(<span class="pl-s"><span class="pl-pds">"</span>/home/huska/pub/tRapTransfac_0.1.tar.gz<span class="pl-pds">"</span></span>)</pre></div>

<h2>
<a id="user-content-developing" class="anchor" href="#developing" aria-hidden="true"><span class="octicon octicon-link"></span></a>Developing</h2>

<h3>
<a id="user-content-git" class="anchor" href="#git" aria-hidden="true"><span class="octicon octicon-link"></span></a>git</h3>

<p>We use the fork &amp; pull model for developement (based on a README from the <a href="https://github.com/shogun-toolbox/shogun/blob/develop/doc/md/README_developer.md">Shogun project</a>)</p>

<p>Visit the tRap github page and click on the <code>Fork</code> button. Then clone your copy of the repo and add the original tRap repository as upstream:</p>

<pre><code>git clone git@github.com:&lt;your id&gt;/tRap.git
git remote add upstream git@github.com:matthuska/tRap.git
git checkout --track origin/master
</code></pre>

<p>Now you're ready to work on your feature. First create a branch:</p>

<pre><code>git checkout -b new_feature_name

</code></pre>

<p>and now you can develop your code and commit to this branch.</p>

<p>Once the feature is done, rebase your branch against the current upstream master:</p>

<pre><code>git fetch upstream
git checkout master
git rebase upstream/master
git checkout new_feature_name
git rebase master
</code></pre>

<p>And now you can push into your repository:</p>

<pre><code>git push
</code></pre>

<p>Now go to the github website and do a <a href="https://help.github.com/articles/using-pull-requests">pull request</a>.</p>

<h3>
<a id="user-content-r-development" class="anchor" href="#r-development" aria-hidden="true"><span class="octicon octicon-link"></span></a>R development</h3>

<p>After creating a branch in your local git repository, modify the R or C++ code however you need and then run R from inside the root of the package's directory (e.g. tRap/) and run a few <code>devtools</code> commands:</p>

<div class="highlight highlight-R"><pre>library(<span class="pl-smi">devtools</span>)
load_all() <span class="pl-c"># this should recompile the C++ code</span>
check()    <span class="pl-c"># does some/all of the above, as well as running R CMD check</span></pre></div>

<p>There are a few other useful <code>devtools</code> commands:</p>

<pre><code>install()  # install the package in your R library directory
document() # just rebuilds the documentation
test()     # just runs all tests
build()    # build a binary package
</code></pre>

<p>For more information check out the <a href="https://github.com/hadley/devtools">devtools</a> site.</p>
