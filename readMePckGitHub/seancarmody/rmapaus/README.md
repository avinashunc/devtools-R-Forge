    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-rmapaus" class="anchor" href="#rmapaus" aria-hidden="true"><span class="octicon octicon-link"></span></a>rmapaus</h1>

<p>R package to simplify visualising Australian data</p>

<p>Here is an example which plots all suburbs falling the in Inner Sydney (10505)
Statistical Division. Note that postcode boundaries do not always align with
Bureau of Statistics boundaries, as is evident here: the black line shows the
border of the 10505 Statistcal Division.</p>

<pre><code>poa &lt;- get_mapaus("POA", accuracy="full")
ssd &lt;- get_mapaus("SSD", accuracy="full")
plot(poa[poa$SSD_CODE11=="10505", ], col="grey", border="white")
with(poa@data[poa$SSD_CODE11=="10505", ], text(long, lat, labels=POA_CODE, cex=0.6))
plot(ssd[ssd$SSD_CODE11=="10505", ], add=TRUE)
</code></pre>

<p><a href="https://camo.githubusercontent.com/671baf7111f21ab395a49cc4b42cf18b1f2944d4/687474703a2f2f692e696d6775722e636f6d2f6f6a35336275472e706e67" target="_blank"><img src="https://camo.githubusercontent.com/671baf7111f21ab395a49cc4b42cf18b1f2944d4/687474703a2f2f692e696d6775722e636f6d2f6f6a35336275472e706e67" alt="Postcode Chart" data-canonical-src="http://i.imgur.com/oj53buG.png" style="max-width:100%;"></a></p>
