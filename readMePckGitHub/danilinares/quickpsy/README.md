    <article class="markdown-body entry-content" itemprop="mainContentOfPage">

<p>quickpsy is an R package developed by <a href="http://www.dlinares.org">Daniel Linares</a> and <a href="http://www.ub.edu/viscagroup/joan/">Joan López-Moliner</a> to quickly fit and plot psychometric functions for multiple conditions. It makes an extensive use of Hadley Wickham's packages <a href="http://ggplot2.org/">ggplot2</a> and <a href="http://cran.r-project.org/web/packages/dplyr/index.html">dplyr</a>.</p>

<p>To understand the fundamentals of fitting psychometric functions in R, we recommend the book <a href="http://www.springer.com/gp/book/9781461444749">Modeling Psychophysical Data in R</a>.</p>

<h2>
<a id="user-content-features" class="anchor" href="#features" aria-hidden="true"><span class="octicon octicon-link"></span></a>Features</h2>

<ul class="task-list">
<li><p>Fits and plots multiple conditions with minimal coding.</p></li>
<li><p>Exploits the computational speed of <a href="http://cran.r-project.org/web/packages/dplyr/index.html">dplyr</a>.</p></li>
<li><p>The user does not need to introduce initial parameters.</p></li>
<li><p>Calculates parametric and non-parametric bootstrap confidence intervals.</p></li>
<li><p>Guess and lapses can be fixed or free as parameters.</p></li>
<li><p>Fits cumulative normal, logistic, weibull functions or any function defined by the user.</p></li>
<li><p>Facilitates the reading of several data files.</p></li>
<li><p>The optimization can call <em>DEoptim</em> which uses differential evolution optimization.</p></li>
</ul>

<h2>
<a id="user-content-install" class="anchor" href="#install" aria-hidden="true"><span class="octicon octicon-link"></span></a>Install</h2>

<p>Download and install <a href="http://cran.rstudio.com">R</a> (we also recommend <a href="http://www.rstudio.com/">Rstudio</a>).</p>

<p>In R, install the following packages: boot, DEoptim, dplyr, ggplot2, tidyr and devtools.</p>

<div class="highlight highlight-r"><pre>install.packages(<span class="pl-s"><span class="pl-pds">'</span>boot<span class="pl-pds">'</span></span>)
install.packages(<span class="pl-s"><span class="pl-pds">'</span>DEoptim<span class="pl-pds">'</span></span>)
install.packages(<span class="pl-s"><span class="pl-pds">'</span>tidyr<span class="pl-pds">'</span></span>)
