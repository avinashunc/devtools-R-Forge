    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h3>
<a id="user-content-overview" class="anchor" href="#overview" aria-hidden="true"><span class="octicon octicon-link"></span></a>Overview</h3>

<p><a href="http://louhos.github.com/sorvi">sorvi</a> is a Finnish open government
  data toolkit for <a href="http://www.r-project.org">R</a>. The package provides
  convenient tools to access, analyze and visualize Finnish data
  resources.</p>

<p>For further details (in Finnish), see the project homepage:
  <a href="http://louhos.github.com/sorvi">http://louhos.github.com/sorvi</a></p>

<h3>
<a id="user-content-contact" class="anchor" href="#contact" aria-hidden="true"><span class="octicon octicon-link"></span></a>Contact</h3>

<p><a href="http://louhos.github.com/contact.html">Homepage</a></p>

<p>You are welcome to:</p>

<ul class="task-list">
<li>submit suggestions and bug-reports at: <a href="https://github.com/louhos/sorvi/issues">https://github.com/louhos/sorvi/issues</a>
</li>
<li>send a pull request on: <a href="https://github.com/louhos/sorvi/">https://github.com/louhos/sorvi/</a>
</li>
<li>compose a friendly e-mail to: <a href="mailto:louhos@googlegroups.com">louhos@googlegroups.com</a>
</li>
<li>join our IRC channels at !louhos@IRCnet (Finland) and ropengov@Freenode (international)</li>
</ul>
