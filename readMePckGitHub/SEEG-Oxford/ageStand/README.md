    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-agestand" class="anchor" href="#agestand" aria-hidden="true"><span class="octicon octicon-link"></span></a>ageStand</h1>

<p><strong>ageStand</strong> is an R package containing functions to convert malaria parasite rate estimates between age ranges. At present this package has only one exported function <code>convertPrevalence</code> which uses the modified Pull and Grab model introduced in Smith <em>et al.</em> (2007) to convert prevalence estimates for <em>Plasmodium falciparum</em> and <em>P. vivax</em>.</p>

<h3>
<a id="user-content-installing-agestand" class="anchor" href="#installing-agestand" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installing ageStand</h3>

<p>To install ageStand you can use the <code>install_github</code> function in the <a href="http://cran.r-project.org/web/packages/devtools/index.html"><code>devtools</code></a> package, like this:</p>

<div class="highlight highlight-r"><pre><span class="pl-c"># install devtools if you haven't already</span>
install.packages(<span class="pl-s"><span class="pl-pds">'</span>devtools<span class="pl-pds">'</span></span>)

<span class="pl-c"># load the package</span>
library(<span class="pl-smi">devtools</span>)

<span class="pl-c"># install ageStand from GitHub</span>
install_github(<span class="pl-s"><span class="pl-pds">'</span>ageStand<span class="pl-pds">'</span></span>, <span class="pl-s"><span class="pl-pds">'</span>SEEG-Oxford<span class="pl-pds">'</span></span>)

<span class="pl-c"># and load it</span>
library(<span class="pl-smi">ageStand</span>)</pre></div>

<h3>
<a id="user-content-using-agestand" class="anchor" href="#using-agestand" aria-hidden="true"><span class="octicon octicon-link"></span></a>Using ageStand</h3>

<p>The package currently contains one function, <code>convertPrevalence</code>, which simplifies conversion of prevalence estimates between age bounds. The following code converts a <em>P. falciparum</em> prevalence estimate for the 2-10 age range to an all-ages prevalence estimate.</p>

<div class="highlight highlight-r"><pre>convertPrevalence(<span class="pl-v">prevalence</span> <span class="pl-k">=</span> <span class="pl-c1">0.1</span>,
                  <span class="pl-v">age_min_in</span> <span class="pl-k">=</span> <span class="pl-c1">2</span>,
                  <span class="pl-v">age_max_in</span> <span class="pl-k">=</span> <span class="pl-c1">9</span>,
                  <span class="pl-v">age_min_out</span> <span class="pl-k">=</span> <span class="pl-c1">0</span>,
                  <span class="pl-v">age_max_out</span> <span class="pl-k">=</span> <span class="pl-c1">85</span>)</pre></div>

<p>This functions can also be vectorised and applied to <em>P. vivax</em>. For full details see the helpfile for this function by typing <code>?convertPrevalence</code> (after installing and loading the package) in R.</p>
