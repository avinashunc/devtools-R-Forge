<h2>
<a id="user-content-purpose" class="anchor" href="#purpose" aria-hidden="true"><span class="octicon octicon-link"></span></a>Purpose</h2>
<p>deathstar is a distributed computing tool for R</p>
<a name="user-content-features"></a>
<h2>
<a id="user-content-features" class="anchor" href="#features" aria-hidden="true"><span class="octicon octicon-link"></span></a>Features</h2>
<p>deathstar allows distributed lapply into Amazon's EC2 cloud or across local compute nodes</p>
<ul>
<li>dead simple</li>
<li>allows both local and remote compute nodes to be used simultaneously</li>
</ul>
<a name="user-content-usage"></a>
<h2>
<a id="user-content-usage" class="anchor" href="#usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Usage</h2>
<p>A minimal example of parallel remote execution.:</p>
<pre>
require(deathstar)

estimatePi &lt;- function(seed) {
    set.seed(seed)
    numDraws &lt;- 1e5
    r &lt;- .5
    x &lt;- runif(numDraws, min=-r, max=r)
    y &lt;- runif(numDraws, min=-r, max=r)
    inCircle &lt;- ifelse( (x^2 + y^2)^.5 &lt; r , 1, 0)
    sum(inCircle) / length(inCircle) * 4
}

cluster &lt;- c("krypton","node1","mongodb","xenon","research")
run.time &lt;-
    system.time(ans &lt;-
                zmq.cluster.lapply(cluster=cluster,
                                   as.list(1:1e3),
                                   estimatePi))

print(mean(unlist(ans)))
print(run.time)
print(attr(ans,"execution.report"))
</pre>
