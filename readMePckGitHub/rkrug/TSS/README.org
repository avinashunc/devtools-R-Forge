<span class="pl-c">##' Basic and extended True Skills Statistics (TSS)</span>
<span class="pl-c">##'</span>
<span class="pl-c">##' TSS is an index of the goodness-of- fit allowing to assess a how</span>

<a id="user-content-51-tss" class="anchor" href="#51-tss" aria-hidden="true"><span class="octicon octicon-link"></span></a>5.1 TSS</h2>Extending the TSS to two continuous input variables.  The reasoning is
  to see if the prediction of presence/absence could be improved by
  using two input variables, i.e. including interaction effects.
<div class="highlight highlight-R"><pre>
<span class="pl-c">##' True skill statistic (TSS) Function</span>
<span class="pl-c">##' </span>
<span class="pl-c">##' TSS is an index of the goodness-of- fit allowing to assess a how</span>
<span class="pl-c">##' much a continuous variable simulated by any model fit</span>
<span class="pl-c">##' presence/absence data.  It thus allows to translate a map of any</span>
<span class="pl-c">##' continuous variable into a map of presence/absence for a given set</span>
<span class="pl-c">##' of spatial simulation at the species level.  This is made possible</span>
<span class="pl-c">##' by finding the threshold value of the continuous variable used as</span>
<span class="pl-c">##' a procxy of fitness (e.g. growth, productivity..).</span>
<span class="pl-c">##'</span>
<span class="pl-c">##' This version extends the function \code{TSS} to using two</span>
<span class="pl-c">##' continuous variables (\code{cont1} and \code{cont2}) and two</span>
<span class="pl-c">##' thresholds (\code{thres1} and \code{thresh2}) to predict \code{presAbs}</span>
<span class="pl-c">##'</span>
<span class="pl-c">##' If \code{largerPres1==TRUE} and code{largerPres2==TRUE} the system</span>
<span class="pl-c">##' represents facilitation, while \code{largerPres1==TRUE} and</span>
<span class="pl-c">##' code{largerPres2==FALSE} represents competition.</span>
<span class="pl-c">##'</span>
<span class="pl-c">##' If \code{is.null(cont2)} the function returns the normal default</span>
<span class="pl-c">##' TSS.</span>
<span class="pl-c">##' </span>
<span class="pl-c">##' Note that \code{cont1} and \code{verref} must match spatially (same ranking and</span>
<span class="pl-c">##' same size)</span>
<span class="pl-c">##' </span>
<span class="pl-c">##' @title True skill statistic</span>
<span class="pl-c">##' @param cont1, \code{vector} of simulation of any continuous variable</span>
<span class="pl-c">##' which has been choosen as a primary proxy of fitness</span>
<span class="pl-c">##' @param cont2 \code{vector} of simulation of any continuous variable</span>
<span class="pl-c">##' which has been choosen as a secondary proxy of fitness. If \code{is.null(cont2)}, a simple TSS is calculated.</span>
<span class="pl-c">##' @param presAbs \code{vector} of presence/absence data, can be logical</span>
<span class="pl-c">##' @param thresh1 \code{vector} containig the threshold values for \code{cont1} </span>
<span class="pl-c">##' @param thresh2 \code{vector} containig the threshold values for \code{cont2}</span>
<span class="pl-c">##' @param largerPres1 if \code{TRUE}, \code{cont1} values larger then</span>
<span class="pl-c">##' the threshold specify presence, if \code{FALSE} \code{cont1}</span>
<span class="pl-c">##' values smaller. </span>
<span class="pl-c">##' @param largerPres2 if \code{TRUE}, \code{cont2} values larger then</span>
<span class="pl-c">##' the threshold specify presence, if \code{FALSE} \code{cont2}</span>
<span class="pl-c">##' values smaller. </span>
<span class="pl-c">##' @param link a \bold{vectorised} function returning a</span>
<span class="pl-c">##' \code{logical} vector of the same length as the \bold{exactly two}</span>
<span class="pl-c">##' \code{logical} input vectors. The default is the function</span>
<span class="pl-c">##' \code{&amp;}. It has to be specified using backquotes (i.e. "`&amp;`" or</span>
<span class="pl-c">##' "`|`").</span>
<span class="pl-c">##' @return An object of class \code{TSS} TODO</span>
<span class="pl-c">##' @author Rainer M. Krug \email{Rainer@@krugs.de}</span>
<span class="pl-c">##'</span>
<span class="pl-c">##' @export</span>
<span class="pl-en">TSS</span> <span class="pl-k">&lt;-</span> <span class="pl-k">function</span>(
    <span class="pl-smi">cont1</span>,
    <span class="pl-v">cont2</span> <span class="pl-k">=</span> <span class="pl-c1">NULL</span>,
    <span class="pl-smi">presAbs</span>,
    <span class="pl-smi">thresh1</span>,
    <span class="pl-smi">thresh2</span>,
    <span class="pl-v">largerPres1</span> <span class="pl-k">=</span> <span class="pl-c1">TRUE</span>,
    <span class="pl-v">largerPres2</span> <span class="pl-k">=</span> <span class="pl-c1">FALSE</span>,
    <span class="pl-v">link</span> <span class="pl-k">=</span> `<span class="pl-k">&amp;</span>`
    ) {
    <span class="pl-smi">presAbs</span> <span class="pl-k">&lt;-</span> as.logical(<span class="pl-smi">presAbs</span>)
    <span class="pl-k">if</span> (is.null(<span class="pl-smi">cont2</span>)) {
        <span class="pl-smi">sel</span> <span class="pl-k">&lt;-</span> (<span class="pl-k">!</span>is.na(<span class="pl-smi">cont1</span>)) <span class="pl-k">&amp;</span> (<span class="pl-k">!</span>is.na(<span class="pl-smi">presAbs</span>))
        <span class="pl-smi">cont1</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">cont1</span>[<span class="pl-smi">sel</span>]
        <span class="pl-smi">presAbs</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">presAbs</span>[<span class="pl-smi">sel</span>]
        <span class="pl-smi">thresh2</span> <span class="pl-k">&lt;-</span> <span class="pl-c1">NA</span>
    } <span class="pl-k">else</span> {
        <span class="pl-smi">sel</span> <span class="pl-k">&lt;-</span> (<span class="pl-k">!</span>is.na(<span class="pl-smi">cont1</span>)) <span class="pl-k">&amp;</span> (<span class="pl-k">!</span>is.na(<span class="pl-smi">cont2</span>)) <span class="pl-k">&amp;</span> (<span class="pl-k">!</span>is.na(<span class="pl-smi">presAbs</span>))
        <span class="pl-smi">cont1</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">cont1</span>[<span class="pl-smi">sel</span>]

        <span class="pl-smi">presAbs</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">presAbs</span>[<span class="pl-smi">sel</span>]
    }
    <span class="pl-smi">thresh</span> <span class="pl-k">&lt;-</span> expand.grid(<span class="pl-smi">thresh1</span>, <span class="pl-smi">thresh2</span>)
    <span class="pl-c">##</span>
    <span class="pl-smi">n</span> <span class="pl-k">&lt;-</span> sum(<span class="pl-smi">presAbs</span>) <span class="pl-k">+</span> sum(<span class="pl-k">!</span><span class="pl-smi">presAbs</span>)
    <span class="pl-smi">n2</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">n</span><span class="pl-k">^</span><span class="pl-c1">2</span>

    <span class="pl-smi">no</span> <span class="pl-k">&lt;-</span> rep(<span class="pl-c1">NA</span>, nrow(<span class="pl-smi">thresh</span>))
    <span class="pl-smi">result</span> <span class="pl-k">&lt;-</span> <span class="pl-k">list</span>(
        <span class="pl-v">threshold1</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">threshold2</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">overallAccuracy</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">sensitivity</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">specificity</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">tss</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">kappa</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">pP</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">pA</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">aA</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">aP</span> <span class="pl-k">=</span> <span class="pl-smi">no</span>,
        <span class="pl-v">n</span> <span class="pl-k">=</span> <span class="pl-smi">n</span>
        )
    
    
    <span class="pl-k">for</span> ( <span class="pl-smi">i</span> <span class="pl-k">in</span> <span class="pl-c1">1</span><span class="pl-k">:</span>nrow(<span class="pl-smi">thresh</span>)) {
        <span class="pl-c">## Classify cont1 and cont2 into absence (&lt; tr)</span>
        <span class="pl-c">## and presence (&gt;= tr) depending on largerPres1</span>
        <span class="pl-c">## and largerPres2</span>
        <span class="pl-smi">result</span><span class="pl-k">$</span><span class="pl-smi">threshold1</span>[<span class="pl-smi">i</span>] <span class="pl-k">&lt;-</span> <span class="pl-smi">thresh</span>[<span class="pl-smi">i</span>, <span class="pl-c1">1</span>]
        <span class="pl-k">if</span> (<span class="pl-smi">largerPres1</span>) {
            <span class="pl-smi">vecCont1</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">cont1</span> <span class="pl-k">&gt;</span><span class="pl-k">=</span> <span class="pl-smi">thresh</span>[<span class="pl-smi">i</span>, <span class="pl-c1">1</span>]
        } <span class="pl-k">else</span> {
            <span class="pl-smi">vecCont1</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">cont1</span> <span class="pl-k">&lt;</span><span class="pl-k">=</span> <span class="pl-smi">thresh</span>[<span class="pl-smi">i</span>, <span class="pl-c1">1</span>]                       
        }
        <span class="pl-k">if</span> (<span class="pl-k">!</span>is.null(<span class="pl-smi">cont2</span>)) {
            <span class="pl-k">if</span> (<span class="pl-smi">largerPres2</span>) {
                <span class="pl-smi">vecCont2</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">cont2</span> <span class="pl-k">&gt;</span><span class="pl-k">=</span> <span class="pl-smi">thresh</span>[<span class="pl-smi">i</span>, <span class="pl-c1">2</span>]
            } <span class="pl-k">else</span> {
                <span class="pl-smi">vecCont2</span> <span class="pl-k">&lt;-</span> <span class="pl-smi">cont2</span> <span class="pl-k">&lt;</span><span class="pl-k">=</span> <span class="pl-smi">thresh</span>[<span class="pl-smi">i</span>, <span class="pl-c1">2</span>]                   
            }
            <span class="pl-c">## Combine vecCont1 and vecCont2 using "link"</span>

        }
