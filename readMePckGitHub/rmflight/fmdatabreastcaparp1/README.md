<h1>
<a id="user-content-fondufe-mittendorf-breast-cancer-parp1-data-package" class="anchor" href="#fondufe-mittendorf-breast-cancer-parp1-data-package" aria-hidden="true"><span class="octicon octicon-link"></span></a>Fondufe-Mittendorf Breast Cancer PARP1 Data Package</h1>

<p>This package contains the data used for the correlation analysis in the publication:</p>

<p><strong>Genome-wide profiling of PARP1 reveals an interplay with gene regulatory regions and DNA methylation</strong>, Nalabothula Narasimharao, Taha Al-jumaily, Robert M. Flight, Shao Xiaorong, Hunter N. B. Moseley, F. Lisa Barcellos, Yvonne Fondufe-Mittendorf, Submitted</p>

<p>This data is used by the <a href="https://github.com/rmflight/fmcorrelationbreastcaparp1">correlation analysis package</a> to generate the vignettes that report the actual correlations between PARP1 reads and other data. Unless otherwise noted, all data was downloaded for the MCF-7 cell line.</p>

<p>Please see the vignette for details on sources of data and processing. In most cases, what is done is simply to take data from UCSC and generate a <code>GenomicRanges</code> object that can be worked with in the analysis.</p>

<p>This package contains the following <code>RData</code> objects:</p>

<ul class="task-list">
<li>
<code>parp1_ln4_reads_all</code>: all PARP1 reads from MCF-7</li>
<li>
<code>parp1_ln5_reads_all</code>: all PARP1 reads from MDA-MB231</li>
<li>
<code>parp1_ln4_unique</code>: summarized reads, with counts at locations capped at 6</li>
<li>
<code>parp1_ln5_unique</code>: summarized reads, with counts at locations capped at 6</li>
<li>
<code>ctcf_rep1</code>: CTCF ChIP-Seq peaks, replicate 1</li>
<li>
<code>ctcf_rep2</code>: CTCF ChIP-Seq peaks, replicate 2</li>
<li>
<code>methyl_rep1</code>: methylation reads, replicate 1</li>
<li>
<code>methyl_rep2</code>: methylation reads, replicate 2</li>
<li>
<code>histone_marks</code>: <code>GRangesList</code> of ChIP-Seq peaks of various histone marks</li>
<li>
<code>expr_data</code>: Affymetrix HGU-133 2.0 expression data</li>
<li>
<code>tss_windows</code>: 2KB windows around Ensembl transcrip start sites</li>
</ul>

<a id="user-content-citation" class="anchor" href="#citation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Citation</h2>

<p>If this package is used in other analyses, it should be cited as:</p>

<p><strong>fmdatabreastcaparp1: Fondufe-Mittendorfe lab data for PARP1 correlation in MCF-7 and MDA-MB231</strong>, R. M. Flight, T. Al-jumaily, Y. Fondufe-Mittendorf, H. N. B. Moseley  doi:10.6084/m9.figshare.1267545</p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>To install this package, you will need to first download the data files from <code>figshare</code>, and then clone this package and install:</p>

<pre><code>git clone https://github.com/rmflight/fmdatabreastcaparp1.git
mkdir fmdatabreastcaparp1/data
wget https://dl.dropbox.com/s/89xgvje4rfl0lml/fmdatabreastcaparp1.zip
unzip fmdatabreastcaparp1.zip -d fmdatabreastcaparp1/data

R
library(devtools)
install("fmdatabreastcaparp1")
</code></pre>

<h2>
<a id="user-content-notes-on-installation-and-running-vignette" class="anchor" href="#notes-on-installation-and-running-vignette" aria-hidden="true"><span class="octicon octicon-link"></span></a>Notes on Installation and Running Vignette</h2>

<p>This package requires a fair amount of memory for the vignette to be run. As far as I can tell, it will need ~4GB to run the part of the vignette that determines what to cap the reads at.</p>

<p>Additionally, I have not been able to do more than a <code>build</code> and <code>install</code> on Travis-ci. However, the package does pass <code>R CMD check</code> on my own machine.</p>
