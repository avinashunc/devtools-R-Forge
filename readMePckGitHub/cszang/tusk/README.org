    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-tusk" class="anchor" href="#tusk" aria-hidden="true"><span class="octicon octicon-link"></span></a>tusk</h1>
<p>Make working with gridded climate data in R a little easier.</p>
<h2>
<a id="user-content-purpose" class="anchor" href="#purpose" aria-hidden="true"><span class="octicon octicon-link"></span></a>Purpose</h2>
<p>Extract point-specific climate data from some commonly used gridded
  climate data sets. For CRU temperature and precipitation,
  downscaling to 30 arc seconds using Worldclim data can be
  performed, following the methodology by Mosier et al. 2014 (Mosier,
  T. M. et al. 2014. 30-Arcsecond monthly climate surfaces with
  global land coverage. - Int. J. Climatol. 34: 2175–2188).</p>
<h2>
<a id="user-content-data-sets" class="anchor" href="#data-sets" aria-hidden="true"><span class="octicon octicon-link"></span></a>Data sets</h2>
<p>The data sets have to be downloaded by the user and saved locally
  as netCDF files, or as generic grids (bil-files) in the case of
  Worldclim data, if downscaling shall be performed. Currently, these
  data sets are supported:</p>
<ul>
  <li><a href="http://badc.nerc.ac.uk/view/badc.nerc.ac.uk__ATOM__ACTIVITY_3ec0d1c6-4616-11e2-89a3-00163e251233">CRU TS 3.20</a></li>
  <li><a href="http://digital.csic.es/handle/10261/72264">SPEIbase v.2.2</a></li>
  <li><a href="http://www.cgd.ucar.edu/cas/catalog/climind/pdsi.html">scPDSI from Dai 2011a and 2011b</a></li>
</ul>
<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>
<div class="highlight highlight-R"><pre>
install.packages(<span class="pl-s"><span class="pl-pds">"</span>devtools<span class="pl-pds">"</span></span>)
library(<span class="pl-smi">devtools</span>)
install_github(<span class="pl-s"><span class="pl-pds">"</span>tusk<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>znag<span class="pl-pds">"</span></span>)</pre></div>
<h2>
<a id="user-content-usage" class="anchor" href="#usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Usage</h2>
