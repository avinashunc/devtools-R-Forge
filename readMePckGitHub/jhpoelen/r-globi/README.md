    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1></h1>

<p><a href="https://travis-ci.org/ropensci/rglobi"><img src="https://camo.githubusercontent.com/8276acabcc34a71e12ff76ee429b48b306f755fc/68747470733a2f2f7472617669732d63692e6f72672f726f70656e7363692f72676c6f62692e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/ropensci/rglobi.svg?branch=master" style="max-width:100%;"></a></p>

<p>R library to access species interaction data of <a href="http://globalbioticinteractions.org">http://globalbioticinteractions.org</a></p>

<p>To install <code>rglobi</code> from <a href="http://cran.r-project.org/web/packages/rglobi/">CRAN</a>:</p>

<div class="highlight highlight-R"><pre>install.packages(<span class="pl-s"><span class="pl-pds">"</span>rglobi<span class="pl-pds">"</span></span>)</pre></div>

<p>Or install development version:</p>

<div class="highlight highlight-R"><pre>install.packages(<span class="pl-s"><span class="pl-pds">"</span>devtools<span class="pl-pds">"</span></span>)
<span class="pl-e">devtools</span><span class="pl-k">::</span>install_github(<span class="pl-s"><span class="pl-pds">"</span>ropensci/rglobi<span class="pl-pds">"</span></span>)</pre></div>

<h2>
<a id="user-content-getting-data" class="anchor" href="#getting-data" aria-hidden="true"><span class="octicon octicon-link"></span></a>Getting Data</h2>

<div class="highlight highlight-R"><pre>library(<span class="pl-smi">rglobi</span>)
<span class="pl-c"># find all unique prey names of Homo sapiens</span>
prey_of(<span class="pl-s"><span class="pl-pds">"</span>Homo sapiens<span class="pl-pds">"</span></span>)<span class="pl-k">$</span><span class="pl-smi">target_taxon_name</span>
<span class="pl-c"># is a shortcut of</span>
get_interactions_by_taxa(<span class="pl-v">sourcetaxon</span><span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">'</span>Homo sapiens<span class="pl-pds">'</span></span>, <span class="pl-v">interactiontype</span><span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">'</span>preysOn<span class="pl-pds">'</span></span>)<span class="pl-k">$</span><span class="pl-smi">target_taxon_name</span>

<span class="pl-c"># list of supported interactions types</span>
get_interaction_types()

<span class="pl-c"># all known prey names and locations (latitude, longitude) where birds (Aves) preyed on rodents (Rodentia) in California</span>
<span class="pl-smi">obs</span> <span class="pl-k">&lt;-</span> get_interactions_by_taxa(<span class="pl-v">sourcetaxon</span> <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>Aves<span class="pl-pds">"</span></span>, <span class="pl-v">bbox</span><span class="pl-k">=</span>c(<span class="pl-k">-</span><span class="pl-c1">125.53344800000002</span>,<span class="pl-c1">32.750323</span>,<span class="pl-k">-</span><span class="pl-c1">114.74487299999998</span>,<span class="pl-c1">41.574361</span>), <span class="pl-v">targettaxon</span> <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>Rodentia<span class="pl-pds">"</span></span>, <span class="pl-v">returnobservations</span><span class="pl-k">=</span><span class="pl-c1">T</span>)
<span class="pl-smi">locations</span> <span class="pl-k">&lt;-</span> cbind(<span class="pl-smi">obs</span><span class="pl-k">$</span><span class="pl-smi">target_taxon_name</span>, <span class="pl-smi">obs</span><span class="pl-k">$</span><span class="pl-smi">latitude</span>, <span class="pl-smi">obs</span><span class="pl-k">$</span><span class="pl-smi">longitude</span>)</pre></div>

<p>Please see R help pages (e.g. <code>?get_interactions_by_taxa</code> and <a href="http://cran.r-project.org/web/packages/rglobi/">vignettes</a> for more information.</p>

<h2>
<a id="user-content-tests" class="anchor" href="#tests" aria-hidden="true"><span class="octicon octicon-link"></span></a>Tests</h2>

<p>Tests can be executed using devtools package.</p>

<div class="highlight highlight-R"><pre><span class="pl-c"># workdir should be rglobi repo root directory (check with getwd())</span>
<span class="pl-e">devtools</span><span class="pl-k">::</span>test()</pre></div>

<p>This should reload the library, executes the test_that testcases and show test reports.</p>

<h2>
<a id="user-content-documentation" class="anchor" href="#documentation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Documentation</h2>

<p>roxygen2 is used to generate .Rd and NAMESPACE by running:</p>

<div class="highlight highlight-R"><pre> library(<span class="pl-smi">roxygen2</span>)
 roxygenize(<span class="pl-s"><span class="pl-pds">"</span>.<span class="pl-pds">"</span></span>)</pre></div>

<p>Vignettes are generated using <code>knitr</code> and <code>markdown</code> packages.</p>
