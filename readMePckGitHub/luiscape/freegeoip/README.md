    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-freegeoip-r-package" class="anchor" href="#freegeoip-r-package" aria-hidden="true"><span class="octicon octicon-link"></span></a>freegeoip R Package</h1>

<p>R package to geolocate IP addresses using <a href="http://freegeoip.net/">http://freegeoip.net/</a> . The package only contains one functon <code>freegeoip</code>. The <code>freegeoip</code> function only takes <code>data.frames</code> (with a single IP column). And it returns a <code>data.frame</code> with the IP addresses geocoded to the maximum granularily available in the <a href="http://www.maxmind.com/">http://www.maxmind.com/</a> IPs database.</p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>You will need the <code>devtools</code> package to install the <code>freegeoip</code> package. </p>

<div class="highlight highlight-r"><pre>  install.packages(<span class="pl-s"><span class="pl-pds">'</span>devtools<span class="pl-pds">'</span></span>)

<h2>
<a id="user-content-known-issues" class="anchor" href="#known-issues" aria-hidden="true"><span class="octicon octicon-link"></span></a>Known Issues</h2>

<p>This package was written when trying to geocode IP addresses from SurveyMonkey. SurveyMonkey provides an export feature that allows you to download your list of subscribers in a CSV format. They collect IP addresses from subscribers by default. I needed a function that geocoded the IP column in the CSV file. That need was the main reason why I created this package. </p>

<p>With that said, the package is really slow. It sends a single IP address at a time. As a result, if you have a large list of IP addresses, you end up making many -- <em>many</em> -- queries. I did it this way in order to properly catch errors -- I was geocoding a very international IP address list with lots of errors. Errors are catched and properly categorized as <code>NA</code>s.</p>
