    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-seqbbs" class="anchor" href="#seqbbs" aria-hidden="true"><span class="octicon octicon-link"></span></a>seqbbs</h1>

<p>R package to accompany paper:</p>

<h2>
<a id="user-content-install" class="anchor" href="#install" aria-hidden="true"><span class="octicon octicon-link"></span></a>Install</h2>

<p>To install the latest (potentially unstable) version, try using <a href="https://github.com/hadley/devtools">devtools</a>:</p>

<pre><code>install.packages("devtools")
library(devtools)
install_github('seqbbs', username = 'metalhelix')
</code></pre>

<h2>
<a id="user-content-use" class="anchor" href="#use" aria-hidden="true"><span class="octicon octicon-link"></span></a>Use</h2>

<p><code>seqbbs</code> function runs main algorithm, returning a SeqBBSData object:</p>

<pre><code>library('seqbbs')

test_filename &lt;- system.file("extdata", "paper.txt", package="seqbbs")
ratios &lt;- read.table(test_filename, header = FALSE)

data &lt;- seqbbs(ratios, window = 20, threshold = 0.7)
</code></pre>

<p>You can plot that data directly using <code>plot_changepoints</code> and/or <code>plot_posteriors</code>:</p>

<pre><code>plot_changepoints(data)
plot_posteriors(data)
</code></pre>

<p>You can also get a dataframe of changepoints, given a particular threshold</p>

<pre><code>results &lt;- changepoints(data)
results

   loci log_mean_ratio mean_ratio confidence_lower_bound confidence_upper_bound
1     0   -1.134965092  0.4553625              0.4508998              0.4598695
