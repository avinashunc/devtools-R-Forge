    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-sclero" class="anchor" href="#sclero" aria-hidden="true"><span class="octicon octicon-link"></span></a>sclero</h1>

<p><strong>An R package to measure growth patterns and align sampling spots in photographs</strong></p>

<p>This is a developmental version of the <em>sclero</em> package providing functions to measure growth patterns and align sampling spots in chronologically deposited materials. The package is intended for the fields of sclerochronology, dentrochronology and geology. This R package is connected to the package author's <a href="http://hdl.handle.net/10037/7152">PhD thesis</a> and further presented in manuscripts under revision. The package is, therefore, subject to changes prior to a final stable version to be submitted to CRAN. Nevertheless, all functions included in the package <em>should</em> work as intended already.</p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>Using the <em>devtools</em> package:</p>

<h2>
<a id="user-content-user-manual" class="anchor" href="#user-manual" aria-hidden="true"><span class="octicon octicon-link"></span></a>User Manual</h2>

<h2>

<h2>
<a id="user-content-dependencies" class="anchor" href="#dependencies" aria-hidden="true"><span class="octicon octicon-link"></span></a>Dependencies</h2>

<p>The <em>sclero</em> package depends on:</p>

<ul class="task-list">
<li>
<a href="https://github.com/davidcsterratt/RImageJROI">RImageJROI</a> package. Used to import ImageJ ROI objects to R.</li>
<li>
<a href="http://cran.r-project.org/web/packages/spatstat/index.html">spatstat</a> package. Used for geometric calculations.</li>
<li>
<a href="https://github.com/hadley/plyr">plyr</a> package. Used for quicker and easier list calculations. This dependency could potentially be removed, but the removal would require rewriting many functions included in the <em>sclero</em> package</li>
</ul>
