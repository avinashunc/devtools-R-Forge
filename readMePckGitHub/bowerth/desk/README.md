    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h2>
<a id="user-content-desktop-environments-for-interactive-analysis-using-r-and-shiny" class="anchor" href="#desktop-environments-for-interactive-analysis-using-r-and-shiny" aria-hidden="true"><span class="octicon octicon-link"></span></a>Desktop environments for interactive analysis using R and Shiny</h2>

<p>This is using <a href="http://www.r-project.org/">R</a> and <a href="http://www.rstudio.com/shiny/">Shiny</a>. The underlying radiant framework has been developed by <a href="http://rady.ucsd.edu/faculty/directory/nijs/">Vincent Nijs</a> (<a href="mailto:vnijs@rady.ucsd.edu">vnijs@rady.ucsd.edu</a>). </p>

<h3>
<a id="user-content-install" class="anchor" href="#install" aria-hidden="true"><span class="octicon octicon-link"></span></a>Install</h3>

<ul class="task-list">
<li>Required: <a href="http://cran.rstudio.com/">R</a>, version 3.0.2 or later</li>
<li>Required: <a href="http://www.rstudio.com/shiny/">Shiny</a>, version 0.8.0.99 or later</li>
<li>Required: A modern browser (e.g., Chrome or Safari)</li>
<li>Suggested: <a href="http://www.rstudio.com/ide/download/">Rstudio</a>
</li>
</ul>

<p>Currently Radiant requires the development version of Shiny. On Windows you will need <a href="http://cran.r-project.org/bin/windows/Rtools/">Rtools</a>. On Mac, make sure you have XCode (free, available in the app store) and the "Command Line Tools for Xcode" (Xcode &gt; Preferences &gt; Downloads).</p>

<p>To install the development version of Shiny follow the steps below:</p>

<pre><code>options(repos = c(CRAN = "http://cran.rstudio.com"))
install.packages('devtools')
require(devtools)
install_github('shiny', 'wch', ref='hidden-state')
require(shiny)
</code></pre>

<p>To install and run the Marketing analytics app in Radiant:</p>

<pre><code>install_github('desk','bowerth')
runApp(system.file("industry", package="desk"))
</code></pre>
