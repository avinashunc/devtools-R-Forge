    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-sartools" class="anchor" href="#sartools" aria-hidden="true"><span class="octicon octicon-link"></span></a>SARTools</h1>

<p>SARTools is a R package dedicated to the differential analysis of RNA-seq data. It provides tools to generate descriptive and diagnostic graphs, to run the differential analysis with one of the well known DESeq2 or edgeR packages and to export the results into easily readable tab-delimited files. It also facilitates the generation of a HTML report which displays all the figures produced, explains the statistical methods and gives the results of the differential analysis. Note that SARTools does not intend to replace DESeq2 or edgeR: it simply provides an environment to go with them. For more details about the methodology behind DESeq2 or edgeR, the user should read their documentations and papers.</p>

<p>SARTools is distributed with two R script templates which use functions of the package. For a more fluid analysis and to avoid possible bugs when creating the final HTML report, the user is encouraged to use them rather than writing a new script.</p>

<h2>
<a id="user-content-how-to-install-sartools" class="anchor" href="#how-to-install-sartools" aria-hidden="true"><span class="octicon octicon-link"></span></a>How to install SARTools?</h2>

<p>In addition to the SARTools package itself, the workflow requires the installation of several packages: DESeq2, edgeR, genefilter, xtable and knitr (all available online, see the dedicated webpages). This current version of SARTools has been developed under R 3.1.2 and with DESeq2 1.6.3, edgeR 3.8.5, genefilter 1.48.1 and knitr 1.7. As a DESeq2 or edgeR update might make the workflow unusable due to modifications on the statistical models, care is recommended when updating these packages.</p>

<p>To install the SARTools package from GitHub, open a R session and:</p>

<ul class="task-list">
<li>install DESeq2, edgeR, genefilter and BiocStyle with <code>source("http://bioconductor.org/biocLite.R")</code> and <code>biocLite(c("DESeq2", "edgeR", "genefilter", "BiocStyle"))</code> (if not installed yet)</li>
<li>install devtools with <code>install.packages("devtools")</code> (if not installed yet)</li>
<li>load the devtools R package with <code>library(devtools)</code>
</li>
<li>run <code>install_github("PF2-pasteur-fr/SARTools", build_vignettes=TRUE)</code>
</li>
</ul>

<h2>
<a id="user-content-how-to-use-sartools" class="anchor" href="#how-to-use-sartools" aria-hidden="true"><span class="octicon octicon-link"></span></a>How to use SARTools?</h2>

<p>A PDF vignette (tutorial.pdf) is available within the package and provides extensive information on the use of SARTools. The user can download it from GitHub or open it with <code>vignette("tutorial", package="SARTools")</code>.</p>
