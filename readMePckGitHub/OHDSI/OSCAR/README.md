    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-achilles" class="anchor" href="#achilles" aria-hidden="true"><span class="octicon octicon-link"></span></a>Achilles</h1>

<p>Automated Characterization of Health Information at Large-scale Longitudinal Evidence Systems (ACHILLES) - descriptive statistics about a OMOP CDM v4 database</p>

<h1>
<a id="user-content-getting-started" class="anchor" href="#getting-started" aria-hidden="true"><span class="octicon octicon-link"></span></a>Getting Started</h1>

<p>(Please review the <a href="https://github.com/OHDSI/Achilles/wiki/Additional-instructions-for-Linux">Achilles Wiki</a> for specific details for Linux)</p>

<ol class="task-list">
<li><p>Make sure you have your data in the <a href="http://omop.org/cdm">OMOP CDM v4 format</a>.</p></li>
<li><p>Make sure that you have Java installed. If you don't have Java already intalled on your computed (on most computers it already is installed), go to <a href="http://java.com">java.com</a> to get the latest version.  (If you have trouble building with rJava below, be sure on Windows that your Path variable includes the path to jvm.dll (Windows Button --&gt; type "path" --&gt; Edit Environmental Variables --&gt; Edit PATH variable, add to end ;C:/Program Files/Java/jre/bin/server) or wherever it is on your system.)</p></li>
<li>
<p>in R, use the following commands to install Achilles:</p>

<div class="highlight highlight-r"><pre>install.packages(<span class="pl-s"><span class="pl-pds">"</span>devtools<span class="pl-pds">"</span></span>)
library(<span class="pl-smi">devtools</span>)
install_github(<span class="pl-s"><span class="pl-pds">"</span>ohdsi/DatabaseConnector<span class="pl-pds">"</span></span>)
install_github(<span class="pl-s"><span class="pl-pds">"</span>ohdsi/SqlRender<span class="pl-pds">"</span></span>)
install_github(<span class="pl-s"><span class="pl-pds">"</span>ohdsi/Achilles<span class="pl-pds">"</span></span>)</pre></div>
</li>
<li>
<p>To run the Achilles analysis, use the following commands in R:</p>

<div class="highlight highlight-r"><pre>library(<span class="pl-smi">Achilles</span>)
<span class="pl-smi">connectionDetails</span> <span class="pl-k">&lt;-</span> createConnectionDetails(<span class="pl-v">dbms</span><span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>sql server<span class="pl-pds">"</span></span>, <span class="pl-v">server</span><span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>server.com<span class="pl-pds">"</span></span>)
<span class="pl-smi">achillesResults</span> <span class="pl-k">&lt;-</span> achilles(<span class="pl-smi">connectionDetails</span>, <span class="pl-s"><span class="pl-pds">"</span>cdm4_inst<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>results<span class="pl-pds">"</span></span>)</pre></div>

<p>"cdm4_inst" and "results" are the names of the schemas holding the CDM data and target results respectively. See the <a href="https://github.com/OHDSI/DatabaseConnector">DatabaseConnector</a> package for details on settings the connection details for your database, for example by typing</p>

<div class="highlight highlight-r"><pre>?<span class="pl-smi">createConnectionDetails</span></pre></div>

<p>Currently "sql server", "oracle", "postgresql", and "redshift" are supported as dbms.</p>
</li>
<li>
<p>To use <a href="https://github.com/OHDSI/AchillesWeb">AchillesWeb</a> to explore the Achilles statistics, you must first export the statistics to JSON files:</p>

<div class="highlight highlight-r"><pre>exportToJson(<span class="pl-smi">connectionDetails</span>, <span class="pl-s"><span class="pl-pds">"</span>cdm4_inst<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>results<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>c:/myPath/AchillesExport<span class="pl-pds">"</span></span>)</pre></div>
</li>
</ol>
