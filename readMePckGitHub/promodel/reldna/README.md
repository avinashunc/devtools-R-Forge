    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-dna-electrostatics-in-r" class="anchor" href="#dna-electrostatics-in-r" aria-hidden="true"><span class="octicon octicon-link"></span></a>DNA Electrostatics in R</h1>

<p>Fast algorithm of calculation of DNA electrostatics was developed in <a href="http://promodel.icb.psn.ru/publab/?q=en/node/5">our laboratory</a> in 1999. Since then we have looked through regulatory elements of many genomes and show that electrostatical properties of regulatory DNA sites could explain a lot of features of DNA-protein recognition process.</p>

<p>All analysis in that papers was done with Matlab, which is not free. Active development of R and Bioconductor project especially make us to migrate our code to R/Bioconductor, so it will be much more accessible to bioinformatics community.</p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>INSTALLATION</h2>

<p>The <strong>reldna</strong> library is pure  R code so it should work on any platform where R is installed.
Installation process is really straightforward:</p>

<ol class="task-list">
<li>Download <a href="https://github.com/promodel/reldna/raw/master/distr/reldna_0.0-9.20130208.tar.gz">latest version</a> of the library</li>
<li>Use standard R installation procedure

<ul class="task-list">
<li>Chechk that all dependencies are installed in your system</li>
<li>Install library </li>
</ul>
</li>
</ol>

<pre><code>R CMD INSTALL reldna_THE_VERSION_YOU_VE_DOWNLOADED.tar.gz
</code></pre>
