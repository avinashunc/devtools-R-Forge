    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-formr" class="anchor" href="#formr" aria-hidden="true"><span class="octicon octicon-link"></span></a>formr</h1>

<h4>
<a id="user-content-the-accompanying-r-package-for-the-survey-framework-formr" class="anchor" href="#the-accompanying-r-package-for-the-survey-framework-formr" aria-hidden="true"><span class="octicon octicon-link"></span></a>The accompanying R package for the survey framework <a href="https://github.com/rubenarslan/formr.org">formr</a>
</h4>

<p>The formr R package provides a few convenience functions that may be useful to the users of formr (<a href="https://formr.org">formr.org</a>), an online survey framework which heavily relies on R via <a href="https://github.com/jeroenooms/opencpu">openCPU</a>. Some functions may be useful to others too.</p>

<p>Some of the functions conveniently generate individual feedback graphics, some are just shorthands to make certain common operations in formr more palatable to R novices. A large number of functions help connect to formr, extract data, correctly type data (dates, numbers, text etc.), automatically aggregate items of a scale and so on.</p>

<p>The R package can be used inside formr.org, it is always loaded by default.</p>

<p>If you want to install the R package locally (e.g. to connect to formr and fetch the data in a nice format for you), run:</p>

<pre><code>install.packages("devtools")
devtools::install_github("rubenarslan/formr")
</code></pre>
