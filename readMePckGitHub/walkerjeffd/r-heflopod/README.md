    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-heflopod-package" class="anchor" href="#heflopod-package" aria-hidden="true"><span class="octicon octicon-link"></span></a>heflopod Package</h1>

<p>Jeffrey D. Walker</p>

<p>This is an R package that does nothing useful. It's sole purpose is to learn about package development.</p>

<h3>
<a id="user-content-set-up" class="anchor" href="#set-up" aria-hidden="true"><span class="octicon octicon-link"></span></a>Set Up</h3>

<ul class="task-list">
<li>Make sure roxygen2 is installed</li>
</ul>

<div class="highlight highlight-R"><pre><span class="pl-e">devtools</span><span class="pl-k">::</span>install_github(<span class="pl-s"><span class="pl-pds">"</span>klutometis/roxygen<span class="pl-pds">"</span></span>)</pre></div>

<ul class="task-list">
<li>Create new package project in RStudio (check the Create git Repository box)</li>
<li>Create repo on Github</li>
<li>Add github as origin</li>
</ul>

<pre><code>git remote add origin git@github.com/walkerjeffd/r-heflopod.git
</code></pre>

<ul class="task-list">
<li>Push master</li>
</ul>

<pre><code>git push -u origin master
</code></pre>

<ul class="task-list">
<li>Add readme file, commit and push to github</li>
</ul>

<h3>
<a id="user-content-building" class="anchor" href="#building" aria-hidden="true"><span class="octicon octicon-link"></span></a>Building</h3>

<ul class="task-list">
<li>Add function(s) in <code>R/</code> folder (e.g. <code>R/cat_function.R</code>)</li>
<li>Add roxygen documentation to each function</li>
<li>(If you build the package now, the package will not export any new functions because they are not added to NAMESPACE)</li>
<li>Compile documentation, which adds the function to NAMESPACE and a *.Rd to man/ for the new function</li>
</ul>

<ul class="task-list">
<li>Install the package manually, which will build it and re-load it in the current session</li>
</ul>

<ul class="task-list">
<li>Or use the RStudio Build &amp; Reload button, which will build and re-load in a <em>new</em> session</li>
<li>Optional: Configure the project to run oxygen automatically on build (see Project options)</li>
<li>Note that roxygen2 does not update the DESCRIPTION file directly. It creates a separate <code>man/heflopod-package.Rd</code> file that is shown via <code>?heflopod</code>. See <a href="http://stackoverflow.com/a/9540038">this stackover answer</a> for details.</li>
