    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-breakthrough" class="anchor" href="#breakthrough" aria-hidden="true"><span class="octicon octicon-link"></span></a>Breakthrough</h1>

<h3>
<a id="user-content-introduction" class="anchor" href="#introduction" aria-hidden="true"><span class="octicon octicon-link"></span></a>Introduction</h3>

<p>This is the final project for course "Microbial Informatics" at the University of Michigan. The goal of this project is develop a program that could be useful.<br>
My idea for project is related to my independent study on Radium removal by ion-exchange resins. I run several column tests, and sample of the effluent. Then the collected samples will run on ICP-MS, which is a type of mass spectrometry that is capable of detecting metals and several non-metals at very low concentrations. When I get the raw data from ICP-MS, I have to plot the data, and calculate the ion-exchange capacity of the resins for different chemial elements. The form of ICP-MS output data is Microsoft Excel. Files include sample number, reaction time, flow rate, dilution1, dilution2 and the concentratoin and RSD of tested chemical elements, which include Na, Mg, Ca, Sr, Ba and Ra. So the objective of this project is to create a program that could plot breakthrough curve and calculate ion-exchange capacity.</p>

<h3>
<a id="user-content-how-to-use" class="anchor" href="#how-to-use" aria-hidden="true"><span class="octicon octicon-link"></span></a>How to use</h3>

<p>This instruction is for Windows.<br>
1.Install packages<br>
Package <code>breakthrough</code> could be downloaded from <a href="https://github.com/hzhang2014">GitHub</a>.       </p>

<div class="highlight highlight-R"><pre><span class="pl-c">####First install `devtools`, a collection of package development tools####</span>
install.packages(<span class="pl-s"><span class="pl-pds">"</span>devtools<span class="pl-pds">"</span></span>)
<span class="pl-c">####load `devtools`####</span>
library(<span class="pl-smi">devtools</span>)
<span class="pl-c">####Install `breakthrough` from github####</span>
install_github(<span class="pl-s"><span class="pl-pds">"</span>final<span class="pl-pds">"</span></span>,<span class="pl-s"><span class="pl-pds">"</span>hzhang2014<span class="pl-pds">"</span></span>)
<span class="pl-c">####Install `xlsx`, a package to read, write and format Excel files####</span>
install.packages(<span class="pl-s"><span class="pl-pds">"</span>xlsx<span class="pl-pds">"</span></span>)
<span class="pl-c">####Install `Hmisc`, a package contains functions for data analysis. This package will be used to generate error bar in the plot.####</span>
install.packages(<span class="pl-s"><span class="pl-pds">"</span>Hmisc<span class="pl-pds">"</span></span>)</pre></div>

<p>2.Load packages</p>

<div class="highlight highlight-R"><pre>library(<span class="pl-smi">breakthrough</span>)
library(<span class="pl-smi">xlsx</span>)
library(<span class="pl-smi">Hmisc</span>)</pre></div>

<p>3.Use packages<br>
There are three functions included in the package:        </p>

<ul class="task-list">
<li>
<code>breakthrough</code> is the function to create breakthrough curve.<br>
<strong>input</strong>: file_name.xlsx<br>
<strong>output</strong>: figure<br>
<strong>Example</strong>: </li>
</ul>

<div class="highlight highlight-R"><pre>breakthrough(<span class="pl-s"><span class="pl-pds">"</span>Radium.xlsx<span class="pl-pds">"</span></span>)</pre></div>

<p><a href="/hzhang2014/final/blob/master/example_breakthrough.png" target="_blank"><img src="/hzhang2014/final/raw/master/example_breakthrough.png" width="500" style="max-width:100%;"></a><br>
<em>Note:</em><br>
To go through with this example, you need to download <a href="https://github.com/hzhang2014/final/blob/master/Radium.xlsx">Radium.xlsx</a> file in your working directory.</p>

<ul class="task-list">
<li>
<code>breakthrough.norm</code> is the function to create a breakthrough curve with normalized x-axis and y-axis.<br>
<strong>input</strong>: file_name.xlsx, bed volume of the resin in <em>ml</em><br>
<strong>output</strong>: figure<br>
<strong>Example</strong>: </li>
</ul>

<div class="highlight highlight-R"><pre>breakthrough.norm(<span class="pl-s"><span class="pl-pds">"</span>Radium.xlsx<span class="pl-pds">"</span></span>,<span class="pl-c1">12</span>)</pre></div>

<p><a href="/hzhang2014/final/blob/master/example_breakthrough_norm.png" target="_blank"><img src="/hzhang2014/final/raw/master/example_breakthrough_norm.png" width="500" style="max-width:100%;"></a></p>

<ul class="task-list">
<li>
<code>capacity</code> is the function to calculate ion-exchange capacity of a certain chemical element.<br>
<strong>input</strong>: file_name.xlsx, weight of resin in <em>g</em>, element interested<br>
<strong>output</strong>: a complete sentense<br>
<strong>Example</strong>: </li>
</ul>

<div class="highlight highlight-R"><pre>capacity(<span class="pl-s"><span class="pl-pds">"</span>Radium.xlsx<span class="pl-pds">"</span></span>,<span class="pl-c1">9.6</span>,<span class="pl-s"><span class="pl-pds">"</span>Ra<span class="pl-pds">"</span></span>)</pre></div>

<div class="highlight highlight-R"><pre>[<span class="pl-c1">1</span>] <span class="pl-s"><span class="pl-pds">"</span>Ion-exchange capacity for Ra is 0.154 nCi/g<span class="pl-pds">"</span></span></pre></div>
