    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-client-for-great-analysis" class="anchor" href="#client-for-great-analysis" aria-hidden="true"><span class="octicon octicon-link"></span></a>Client for GREAT Analysis</h1>

<h2>
<a id="user-content-install" class="anchor" href="#install" aria-hidden="true"><span class="octicon octicon-link"></span></a>Install</h2>

<p><strong>rGREAT</strong> is available on Bioconductor (<a href="http://bioconductor.org/packages/devel/bioc/html/rGREAT.html">http://bioconductor.org/packages/devel/bioc/html/rGREAT.html</a>)</p>

<div class="highlight highlight-r"><pre>source(<span class="pl-s"><span class="pl-pds">"</span>http://bioconductor.org/biocLite.R<span class="pl-pds">"</span></span>)
biocLite(<span class="pl-s"><span class="pl-pds">"</span>rGREAT<span class="pl-pds">"</span></span>)</pre></div>

<h2>
<a id="user-content-usage" class="anchor" href="#usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Usage</h2>

<p>The input data is a <code>GRanges</code> object or a <em>BED</em>-format data frame, no matter it is sorted or not.</p>

<div class="highlight highlight-r"><pre>library(<span class="pl-smi">rGREAT</span>)
set.seed(<span class="pl-c1">123</span>)
<span class="pl-v">bed</span> <span class="pl-k">=</span> <span class="pl-e">circlize</span><span class="pl-k">::</span>generateRandomBed(<span class="pl-v">nr</span> <span class="pl-k">=</span> <span class="pl-c1">1000</span>, <span class="pl-v">nc</span> <span class="pl-k">=</span> <span class="pl-c1">0</span>)
head(<span class="pl-smi">bed</span>)</pre></div>

<pre><code>##    chr    start      end
## 1 chr1   155726  2608935
## 2 chr1  6134977 10483365
## 3 chr1 11354986 11423447
## 4 chr1 15134641 19115321
## 5 chr1 22692774 23328609
## 6 chr1 23639094 25639077
</code></pre>

<p>You can get the summary of your job by calling <code>job</code> variable.</p>
