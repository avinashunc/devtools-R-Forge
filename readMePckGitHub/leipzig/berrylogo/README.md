    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-berrylogo-a-better-sequence-logo" class="anchor" href="#berrylogo-a-better-sequence-logo" aria-hidden="true"><span class="octicon octicon-link"></span></a>berrylogo: a better sequence logo</h1>

<h2>
<a id="user-content-description" class="anchor" href="#description" aria-hidden="true"><span class="octicon octicon-link"></span></a>Description</h2>

<p>A berrylogo is a sequence logo alternative developed by <a href="http://famprevmed.ucsd.edu/faculty/cberry">Charles C.
Berry</a> for "Selection of
Target Sites for Mobile DNA Integration in the Human Genome"
<a href="http://www.ncbi.nlm.nih.gov/pmc/articles/PMC1664696/"></a><a href="http://www.ncbi.nlm.nih.gov/pmc/articles/PMC1664696">http://www.ncbi.nlm.nih.gov/pmc/articles/PMC1664696</a></p>

<p>Instead of “information content”, the y-axis is the log relative
frequency with respect to the background frequency, generated here from
the gc_content parameter.</p>

<h2>
<a id="user-content-usage" class="anchor" href="#usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Usage</h2>

<pre><code>library("devtools")
install_github("leipzig/berrylogo")
library("berrylogo")
freqs&lt;-matrix(data=c(0.25,0.65,0.87,0.92,0.16,0.16,0.04,0.98,0.98,1.00,0.02,0.10,0.10,0.80,0.98,0.91,0.07,0.07,
             0.11,0.05,0.04,0.00,0.26,0.17,0.00,0.01,0.00,0.00,0.29,0.17,0.01,0.03,0.00,0.00,0.32,0.32,
             0.53,0.26,0.07,0.02,0.53,0.18,0.96,0.01,0.00,0.00,0.65,0.01,0.89,0.17,0.01,0.09,0.59,0.12,
             0.11,0.04,0.02,0.06,0.05,0.49,0.00,0.00,0.02,0.00,0.04,0.72,0.00,0.00,0.01,0.00,0.02,0.49),
             byrow=TRUE,nrow=4,dimnames=list(c('A','C','G','T')))
p&lt;-berrylogo(freqs,gc_content=.41)
print(p)
