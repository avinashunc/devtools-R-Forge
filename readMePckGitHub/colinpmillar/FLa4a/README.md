<h2>
<a id="user-content-overview" class="anchor" href="#overview" aria-hidden="true"><span class="octicon octicon-link"></span></a>Overview</h2>

<p>FLa4a implements the Assesment For All (A4A) initiative stock assessment model, a simple and robust statistical catch at age model. For further information about rhe A4A initiative, please visit <a href="https://fishreg.jrc.ec.europa.eu/web/a4a">https://fishreg.jrc.ec.europa.eu/web/a4a</a>.</p>

<p>To install this package, start R and enter:</p>

<pre><code>install.packages(c("copula", "mgcv", "triangle"))
</code></pre>

<p>followed by</p>

<pre><code>install.packages("FLa4a", repos="http://flr-project.org/R")
</code></pre>

<p>or download from the <a href="https://github.com/flr/FLa4a/releases/latest">FLa4a releases page</a></p>
