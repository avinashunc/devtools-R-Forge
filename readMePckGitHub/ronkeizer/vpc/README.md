    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-vpc" class="anchor" href="#vpc" aria-hidden="true"><span class="octicon octicon-link"></span></a>vpc</h1>

<p>Create visual predictive checks, a commonly used diagnostic plot in pharmacometrics, in R </p>

<h2>
<a id="user-content-rationale" class="anchor" href="#rationale" aria-hidden="true"><span class="octicon octicon-link"></span></a>Rationale</h2>

<p>The VPC is a widely used diagnostic tool in pharmacometrics (see e.g. <a href="http://page-meeting.org/default.asp?abstract=1434">here</a>), most commonly created using <a href="http://psn.sourceforge.net">PsN</a> and <a href="http://xpose.sourceforge.net">Xpose</a>, using NONMEM as the simulation engine. The aim of the current library is to provide an improved tool that is:</p>

<ul class="task-list">
<li>a single-step process for creating a VPC in R (not using Xpose or PsN). 

<ul class="task-list">
<li>e.g. allows changing of vpc parameters such as binning / stratification upon creation of the plot, not in a separate pre-processing step.</li>
<li>easier debugging than PsN+Xpose, all data parsing and plotting in one R function</li>
</ul>
</li>
<li>more flexible regarding input (use simulated data from R, NONMEM, Monolix, Stan, or any other simulation tool)</li>
<li>easier to customize, e.g. request any prediction / confidence interval or binning strategy upon plotting.</li>
<li>easier to extend: the output is a ggplot object which can be themed and extended</li>
</ul>

<h2>
<a id="user-content-functionality-available" class="anchor" href="#functionality-available" aria-hidden="true"><span class="octicon octicon-link"></span></a>Functionality available</h2>

<p>Plots:</p>

<ul class="task-list">
<li>VPC for continuous data</li>
<li>VPC for censored continuous data (e.g. below LOQ)</li>
<li>VPC for categorical data</li>
<li>VPC for time-to-event data (single and repeated)</li>
<li>Kaplan-Meier Mean Covariate plots <a href="http://page-meeting.org/pdf_assets/4280-2012-06%20PAGE%20KMMC.pdf">KMMC</a>
</li>
<li>
<em>in progress</em>: npde plots with prediction intervals, see e.g. <a href="http://page-meeting.org/pdf_assets/9164-venise12_posternpde.pdf">here</a>
</li>
</ul>

<p>Options:</p>

<ul class="task-list">
<li>stratification (single &amp; multiple)</li>
<li>prediction-correction</li>
<li>binning methods (K-means clustering, by time, by number of data points, by density, etc).</li>
<li>automatated number of bins (using Sturges' rule) or manual input of number of bins</li>
<li>plot without simulated data (<code>sim=NULL</code>), or without observed data(<code>obs=NULL</code>).</li>
<li>general-purpose function (<code>sim_data</code>) to simulate data from a mixed-effects structural model, a fixed parameter vector and between-subject variability covariance matrix.</li>
</ul>
