    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-sbw" class="anchor" href="#sbw" aria-hidden="true"><span class="octicon octicon-link"></span></a>SBW</h1>

<p>This package arises from the from the NCEAS 2013 animal tracking
working group meeting in Santa Barbara.</p>

<p>The package has three main functions.</p>

<ul class="task-list">
<li><p>The package provides a set of routines of the astronomical
calculations that underpin threshold Geolocation estimates, based on
the javascript code and the excel spreadsheets from the NOAA solar
calculator site (<a href="http://www.esrl.noaa.gov/gmd/grad/solcalc/">http://www.esrl.noaa.gov/gmd/grad/solcalc/</a>).</p></li>
<li><p>The package reproduces some basic functionality of the
<a href="http://cran.r-project.org/web/packages/GeoLight/index.html">GeoLight</a> package
and provides a number of extensions for simulating light records and
examining the sensitivity of threshold based position estimates.</p></li>
<li><p>The package implements threshold based variants of the models
implemented in the
<a href="http://cran.r-project.org/web/packages/tripEstimation/index.html">tripEstimation</a> package.</p></li>
</ul>

<p>It is not expected that this package will ever be formally released
through CRAN. Rather it is hoped that the functionality in this
package will ultimately be integrated back into the GeoLight and
tripEstimation packages.</p>

<h2>
<a id="user-content-installing" class="anchor" href="#installing" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installing</h2>

<p>The package is easily built with RStudio</p>

<ol class="task-list">
<li><p>Install R</p></li>
<li><p>Install <a href="http://www.rstudio.com">RStudio</a></p></li>
<li><p>Install <a href="http://cran.r-project.org/bin/windows/Rtools/">Rtools</a> or equivalent for your platform</p></li>
<li><p>Install <a href="http://cran.r-project.org/bin/windows/Rtools/">devtools</a> and <a href="http://cran.r-project.org/web/packages/roxygen2/index.html">roxygen2</a> packages and dependencies in R</p></li>
<li><p>Install <a href="http://cran.r-project.org/web/packages/geosphere/index.html">geosphere</a>, <a href="http://cran.r-project.org/web/packages/GeoLight/index.html">GeoLight</a> packages and dependencies.</p></li>
<li><p>Clone the repository</p></li>
<li><p>Create an Rstudio project in the folder containing this Readme file.</p></li>
<li><p>In the build tab, choose <code>More/Configure Build Tools...</code> and click

<li><p>Choose <code>More/Roxygenize</code> from the <code>Build</code> tab</p></li>
<li><p>Choose <code>Build &amp; Reload</code> to make the package immediately available to R, or choose <code>More/Build source package</code> <code>More/Build binary package</code> from the <code>Build</code> tab to make packages.</p></li>
</ol>

Unfortunately, at the time of writing there does not seem to be a good parallization solution that works equally well on all platforms.  This is a low priority.</p></li>
