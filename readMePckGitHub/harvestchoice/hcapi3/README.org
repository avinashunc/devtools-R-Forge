    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-hcapi3" class="anchor" href="#hcapi3" aria-hidden="true"><span class="octicon octicon-link"></span></a>HCAPI3</h1>
<p>Port of HarvestChoice Data Services to an OpenCPU server at <a href="http://hcapi.harvestchoice.org/ocpu/library/hcapi3">http://hcapi.harvestchoice.org/ocpu/library/hcapi3</a>. More about HarvestChoice datasets for sub-Saharan Africa at <a href="http://harvestchoice.org/data/">http://harvestchoice.org/data/</a>.</p>
<h2>
<a id="user-content-overview" class="anchor" href="#overview" aria-hidden="true"><span class="octicon octicon-link"></span></a>Overview</h2>
<p>The package documentation is at <a href="http://hcapi.harvestchoice.org/ocpu/library/hcapi3/info">http://hcapi.harvestchoice.org/ocpu/library/hcapi3/info</a>. Documentation for each API method may be accessed using <code>/man/&lt;method&gt;</code>, for example at <a href="http://hcapi.harvestchoice.org/ocpu/library/hcapi3/man/getLayer">http://hcapi.harvestchoice.org/ocpu/library/hcapi3/man/getLayer</a>
  also in PDF format using <code>/man/&lt;method&gt;/pdf</code> at
  <a href="http://hcapi.harvestchoice.org/ocpu/library/hcapi3/man/getLayer/pdf">http://hcapi.harvestchoice.org/ocpu/library/hcapi3/man/getLayer/pdf</a>.</p>
<p>HTTP POST requests may be tested interactively at <a href="http://hcapi.harvestchoice.org/ocpu/">http://hcapi.harvestchoice.org/ocpu/</a>. To test requests in the console, use the following <code>curl</code> syntax. Arguments may be passed as well-structured json array with curl <code>-H Content-Type:application/json</code> flag or more directly as URL arguments using <code>-d param1=value1&amp;param2=value2</code> construct. JavaScript examples are available from <a href="https://www.opencpu.org/jslib.html">OpenCPU</a>.</p>
<div class="highlight highlight-sh"><pre>
<span class="pl-c"># Simple data request with aggregation at district level (Ethiopia)</span>
<span class="pl-c"># returning a JSON data array</span>
$  curl http://hcapi.harvestchoice.org/ocpu/library/hcapi3/R/getLayer/json \
-d <span class="pl-s"><span class="pl-pds">'</span>{"var": ["whea_h", "bmi"], "iso3": "ETH", "by": "ADM2_NAME_ALT", "as.class" : "list"}<span class="pl-pds">'</span></span> \
-X POST -H <span class="pl-s"><span class="pl-pds">"</span>Content-Type:application/json<span class="pl-pds">"</span></span></pre></div>
<div class="highlight highlight-json"><pre>
{
