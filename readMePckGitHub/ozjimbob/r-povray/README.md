    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-r-povray" class="anchor" href="#r-povray" aria-hidden="true"><span class="octicon octicon-link"></span></a>r-povray</h1>

<p>An R Package to interface with the POVRay raytracing system.
<a href="https://camo.githubusercontent.com/068dcc40c35185f967b16cb0a01ea312fffe5e46/687474703a2f2f692e696d6775722e636f6d2f51624a635777782e706e67" target="_blank"><img src="https://camo.githubusercontent.com/068dcc40c35185f967b16cb0a01ea312fffe5e46/687474703a2f2f692e696d6775722e636f6d2f51624a635777782e706e67" alt="Demonstration Image" data-canonical-src="http://i.imgur.com/QbJcWwx.png" style="max-width:100%;"></a></p>

<h2>
<a id="user-content-introduction" class="anchor" href="#introduction" aria-hidden="true"><span class="octicon octicon-link"></span></a>Introduction</h2>

<p>R-povray (R package name <strong>povray</strong>) aims to allow R users to produce 3D rendered graphics in the <a href="http://www.povray.org">POVRay</a> raytracing software.  Initially, classes will be provided to create basic POVRay objects (lights, cameras, primitives, textures).  Further higher-level classes to render graphs will be the ultimate aim.  The page does not aim to provide an interface to <em>all</em> POVRay functions, but does aim to make basic, commonly-used POVRay objects available in an easy to use manner from within R.</p>

<h2>
<a id="user-content-structure" class="anchor" href="#structure" aria-hidden="true"><span class="octicon octicon-link"></span></a>Structure</h2>

<p>The package provides a range of environment-based classes, for example a <strong>Scene</strong> class that brings together the 3D objects, writes POVRay source files and rengers them, a <strong>Light</strong> class that allows placement of lights of a certain colour in a certain location, a <strong>Box</strong> class, a <strong>Camera</strong> class etc.</p>

<p>The image above is created as follows:</p>

<p>Define scene, default background colour</p>

<pre><code>sc=Scene()

<pre><code>green=Color(0,1,0)

<p>Place 30 random green poles with spheres.</p>

<pre><code>for(idx in 1:30){

    cyl = Cylinder(c(x,0,z),c(x,y,z),0.3,col=green,tex=shinyt)
    sph= Sphere(c(x,y,z),1,col=green,tex=shinyt)
    sc$push(cyl)
    sc$push(sph)
}
</code></pre>
