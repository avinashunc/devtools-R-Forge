<h1>diana</h1>
<p>Distraktoren- und Itemanalyse dichotomer sowie nach dem Partial-Credit-Modell kodierter Multiple-Choice-Items mit R.</p>
<h3>Installation</h3>
<p>Auszuführende R-Befehle:</p>
<pre><code class="r">install.packages(&quot;devtools&quot;)
require(&quot;devtools&quot;)
install_github(&quot;diana&quot;, username = &quot;loersch&quot;)
</code></pre>
<h3>Enthaltene Funtionen</h3>
<h4>Hauptfunktionen</h4>
<ul>
<li><strong>alphaComb</strong>: Berechnung des Cohen&#39;s Alpha für verschiedene Item-Bepunktungskombinationen (<code>alpha</code>)</li>
<li><strong>vRecode</strong>: eingeschränkte Rekodierung einzelner Variablen einer Daten-Matrix (<code>recode</code>)</li>
<li><strong>prepare</strong>: Rekodierung der Testdaten (<code>recode</code>, <code>score.multiple.choice</code>)</li>
<li><strong>gTable</strong>: erzeugt pdf-Ausgabe der Funktionen <code>item.exam()</code> und <code>score.multiple.choice()</code> </li>
<li><strong>gPlot</strong>: erzeugt pdt-Ausgabe mit Balkendiagramme sowie Testdatentabellen je Item (<code>item.exam</code>)</li>
<li><strong>rasch</strong>: erzeugt pdf-Ausgabe mit einer Raschanalyse mithilfe der eRm-Erweiterung </li>
<li><strong>piMap</strong>: erzeugt aus den Personen-, Itemparametern der Rasch-Anahalyse eine item-person-map (<code>xtable</code>)</li>
<li><strong>mKappa</strong>: Berechnung der Interrater-Reliabilität für Mehrfachkodierungen (<code>cohen.kappa</code>)</li>
</ul>
<h4>Hilfsfunktionen</h4>
<ul>
<li><strong>str2char</strong>: Umwandlung aller Faktoren eines data.frames in Character</li>
<li><strong>umlauts</strong>: Umwandlung von Umlauten </li>
</ul>
