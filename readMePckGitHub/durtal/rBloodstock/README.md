    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-rbloodstock-010" class="anchor" href="#rbloodstock-010" aria-hidden="true"><span class="octicon octicon-link"></span></a>rBloodstock 0.1.0</h1>

<p>R package with variety of datasets from Thoroughbred Bloodstock Sales, at the time of writing there are just 5 datasets of Tattersalls sales from 2010 to 2014 (incomplete)</p>

<h4>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h4>

<p>Within R</p>

<div class="highlight highlight-R"><pre><span class="pl-e">devtools</span><span class="pl-k">::</span>install_github(<span class="pl-v">repo</span> <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>durtal/rBloodstock<span class="pl-pds">"</span></span>)</pre></div>

<h4>
<a id="user-content-datasets" class="anchor" href="#datasets" aria-hidden="true"><span class="octicon octicon-link"></span></a>Datasets</h4>

<p>Only Tattersalls sales are in the package at the moment, hopefully others will be added in time (sales from Goffs etc.)</p>

<h5>
<a id="user-content-tattersalls-sales" class="anchor" href="#tattersalls-sales" aria-hidden="true"><span class="octicon octicon-link"></span></a>Tattersalls sales</h5>

<p>Data was sourced from <a href="http://db.tattersalls.com/4DCGI/Entry/Index">Tattersalls</a></p>
