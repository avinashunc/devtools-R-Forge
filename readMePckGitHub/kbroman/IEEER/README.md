<h2>
<a id="user-content-ieeer-r-interface-to-the-ieee-xplore-gateway" class="anchor" href="#ieeer-r-interface-to-the-ieee-xplore-gateway" aria-hidden="true"><span class="octicon octicon-link"></span></a>IEEER: R interface to the IEEE Xplore Gateway</h2>

<p><a href="http://www.ieee.org/index.html">IEEE</a> stands for the Institute of
Electrical and Electronic Engineers. It publishes well over 100
peer-reviewed journals from
<a href="http://ieeexplore.ieee.org/xpl/RecentIssue.jsp?punumber=8">IEEE Antennas and Propagation</a>
to
<a href="http://ieeexplore.ieee.org/xpl/RecentIssue.jsp?punumber=23">IEEE Nuclear</a>
to
<a href="http://www.ieee-uffc.org/">IEEE Ultrasonics, Ferroelectrics &amp; Frequency Control Society</a>.</p>

<p>The <a href="https://github.com/ropensci/IEEER">IEEER package</a> is an
<a href="http://www.r-project.org">R</a> interface to the
<a href="http://ieeexplore.ieee.org/gateway/">IEEE Xplore Search Gateway</a>.</p>

<h3>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h3>

<p>The package is not currently available on
<a href="http://cran.r-project.org">CRAN</a>. To install, use
<code>devtools:install_github()</code>, as follows:</p>
