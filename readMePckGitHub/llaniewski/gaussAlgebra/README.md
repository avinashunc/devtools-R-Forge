    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-gaussalg---algebra-of-functions-for-r" class="anchor" href="#gaussalg---algebra-of-functions-for-r" aria-hidden="true"><span class="octicon octicon-link"></span></a>gaussAlg - Algebra of functions for R</h1>

<p><a href="https://travis-ci.org/llaniewski/gaussAlgebra"><img src="https://camo.githubusercontent.com/06dcdce2d080ccd1394a83e7b520d3ce96fbe9d2/68747470733a2f2f7472617669732d63692e6f72672f6c6c616e696577736b692f6761757373416c67656272612e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/llaniewski/gaussAlgebra.svg?branch=master" style="max-width:100%;"></a></p>

<p>Algebra of functions which looks like:</p>

<p>polynomial * exp( -x^2 )</p>

<p>in high dimensions. And lot of other neat stuff.</p>

<p>Example:</p>
