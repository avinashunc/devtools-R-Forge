    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-herring" class="anchor" href="#herring" aria-hidden="true"><span class="octicon octicon-link"></span></a>Herring</h1>

<p>This is an R package that is currently under development.
It contains code and data useful for doing genetic stock identification of 
herring species on the east coast.  </p>

<p>It is currently under development.  </p>

<h2>
<a id="user-content-replicating-our-results" class="anchor" href="#replicating-our-results" aria-hidden="true"><span class="octicon octicon-link"></span></a>Replicating our results</h2>

<p>This repository is designed so that our results can be replicated easily.  Here are the commands that you would use at an R prompt to do that:</p>

<div class="highlight highlight-r"><pre><span class="pl-c">#  If you don't have devtools package, install it: install.packages("devtools")</span>

<span class="pl-c"># install the R-package that is this library</span>
<span class="pl-e">devtools</span><span class="pl-k">::</span>install_github(<span class="pl-s"><span class="pl-pds">"</span>eriqande/herring<span class="pl-pds">"</span></span>)

<span class="pl-c"># install the R package "gpiper" that is also needed</span>
<span class="pl-e">devtools</span><span class="pl-k">::</span>install_github(<span class="pl-s"><span class="pl-pds">"</span>eriqande/gpiper<span class="pl-pds">"</span></span>, <span class="pl-v">ref</span> <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>3d5a6b5888<span class="pl-pds">"</span></span>)


<span class="pl-c"># then source the analysis files like this:</span>
