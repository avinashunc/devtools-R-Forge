    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-evolqg-package" class="anchor" href="#evolqg-package" aria-hidden="true"><span class="octicon octicon-link"></span></a>EvolQG Package</h1>

<p><a href="https://gitter.im/diogro/evolqg?utm_source=badge&amp;utm_medium=badge&amp;utm_campaign=pr-badge&amp;utm_content=badge"><img src="https://camo.githubusercontent.com/da2edb525cde1455a622c58c0effc3a90b9a181c/68747470733a2f2f6261646765732e6769747465722e696d2f4a6f696e253230436861742e737667" alt="Join the chat at https://gitter.im/diogro/evolqg" data-canonical-src="https://badges.gitter.im/Join%20Chat.svg" style="max-width:100%;"></a></p>

<p>Proto package.</p>

<p>Kinda usable</p>

<h1>
<a id="user-content-instalation" class="anchor" href="#instalation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Instalation</h1>

<h2>
<a id="user-content-if-windows" class="anchor" href="#if-windows" aria-hidden="true"><span class="octicon octicon-link"></span></a>If Windows</h2>

<p>First install <a href="http://cran.r-project.org/bin/windows/Rtools/">Rtools</a>, then proceed to devtools instalation instructions.</p>

<p>Be careful to use the right version for your R instalation.</p>

<h2>
<a id="user-content-if-macos-x" class="anchor" href="#if-macos-x" aria-hidden="true"><span class="octicon octicon-link"></span></a>If MacOS X</h2>

<p>If you are using MacOS X, you will probably have to build Rcpp from source. 
You will need to have <a href="https://developer.apple.com/xcode/">Xcode</a> or some other C++ compiler installed.
Check if the instalation suceded typing <code>clang++ --version</code> in the terminal. It might be necessary to add a file called Makevars in a .R folder in your Home directory.</p>

<p>In the terminal type:</p>

<pre><code>mkdir .R
cd .R
touch Makevars
</code></pre>

<p>edit this file (".R/Makevars") and add the following lines:</p>

<pre><code>CC=clang
CXX=clang++
CFLAGS="-mtune=native -g -O3 -Wall -pedantic -Wconversion"
CXXFLAGS="-mtune=native -g -O3 -Wall -pedantic -Wconversion"
FLIBS=-lgfortran
</code></pre>

<p>Then, download the Rcpp source from <a href="http://cran.r-project.org/web/packages/Rcpp/index.html">here</a> (Package source link), then install it in R using:</p>

<div class="highlight highlight-R"><pre><span class="pl-k">&gt;</span> install.packages(<span class="pl-s"><span class="pl-pds">"</span>./Rcpp_0.*.tar<span class="pl-pds">"</span></span>, <span class="pl-v">repos</span> <span class="pl-k">=</span> <span class="pl-c1">NULL</span>, <span class="pl-v">type</span><span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>source<span class="pl-pds">"</span></span>)</pre></div>

<p>Replacing the file name with the proper version number.</p>

<p>Then proceed to devtools instalation instructions.</p>

<h2>
<a id="user-content-using-devtools" class="anchor" href="#using-devtools" aria-hidden="true"><span class="octicon octicon-link"></span></a>Using devtools</h2>

<p>Install <a href="http://www.rstudio.com/projects/devtools/">devtools</a> and <a href="http://cran.r-project.org/web/packages/Rcpp/index.html">Rcpp</a>, then:</p>

<div class="highlight highlight-R"><pre><span class="pl-k">&gt;</span> library(<span class="pl-smi">devtools</span>)
<span class="pl-k">&gt;</span> install_github(<span class="pl-s"><span class="pl-pds">"</span>lem-usp/evolqg<span class="pl-pds">"</span></span>)</pre></div>
