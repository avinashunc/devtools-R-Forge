<h2>
<a id="user-content-historydata-data-sets-for-historians" class="anchor" href="#historydata-data-sets-for-historians" aria-hidden="true"><span class="octicon octicon-link"></span></a>historydata: Data Sets for Historians</h2>

<p>These sample data sets are intended for historians learning R. They
include population, institutional, religious, military, and
prosopographical data suitable for mapping, quantitative analysis, and
network analysis.</p>

<h3>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h3>

<p>To install the package from CRAN:</p>

<pre><code>install.packages("historydata")
</code></pre>

<p>To install the development version, you will first have to install
<a href="https://github.com/hadley/devtools">devtools</a> and then install this package from GitHub.</p>

<p>To install:</p>

<pre><code>devtools::install_github("ropensci/historydata")
</code></pre>

<h3>
<a id="user-content-use" class="anchor" href="#use" aria-hidden="true"><span class="octicon octicon-link"></span></a>Use</h3>

<p>To list all the datasets in the package with their documentation:</p>

<pre><code>library(historydata)
help(package = historydata)
</code></pre>

<p>To load a dataset:</p>

<pre><code>data(catholic_dioceses)
</code></pre>

<h3>
<a id="user-content-contributing" class="anchor" href="#contributing" aria-hidden="true"><span class="octicon octicon-link"></span></a>Contributing</h3>

<p>If you have a dataset that you think would be good for this package,
feel free to contribute it. You can send the dataset via e-mail if you
can provide a citation and guarantee that the data is available under an
open license.</p>

<p>But it is much preferred that you contribute the dataset via a pull
request. To add a dataset:</p>

<ol class="task-list">
<li> Add the raw data and an R script to load and save it as an R data
object to <code>data-raw/</code>. The <code>.rda</code> file should be saved to <code>data/</code>.
See <code>data-raw/sarna.R</code> as a model. Please try keeping the data
<a href="http://www.jstatsoft.org/v59/i10">tidy</a>.</li>
<li> Add the documentation using the <a href="http://roxygen.org/">Roxygen</a> format to a file in the
<code>R/</code>. Use <code>R/sarna.R</code> as a model. Be sure to include a citation.</li>
</ol>
