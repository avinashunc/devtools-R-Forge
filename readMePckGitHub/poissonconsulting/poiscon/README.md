    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-poiscon" class="anchor" href="#poiscon" aria-hidden="true"><span class="octicon octicon-link"></span></a>poiscon</h1>

<p>poiscon is an R package of utility functions to simplify date time manipulations
and data, plot, table and analysis saving and loading. It is used by the 
employees of Poisson Consulting Ltd.</p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>To install the latest version of poiscon:</p>

<pre><code># install.packages("devtools")
library(devtools)
install_github("poissonconsulting/tulip")
install_github("poissonconsulting/datalist")
install_github("poissonconsulting/juggler")
install_github("poissonconsulting/jaggernaut")
install_github("poissonconsulting/poiscon")
library(poiscon)
</code></pre>
