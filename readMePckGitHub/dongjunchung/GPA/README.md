    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-gpa" class="anchor" href="#gpa" aria-hidden="true"><span class="octicon octicon-link"></span></a>GPA</h1>

<p>GPA is a statistical approach to prioritizing GWAS results by integrating pleiotropy information and annotation data. 'GPA' package provides computationally efficient and user friendly interface to fit GPA models and implement the hypothesis testing for the pleiotropy and the enrichment of annotation for the associated SNPs.</p>

<p>The stable versions of GPA package can be obtained from the following URLs:</p>

<p>Package source: <a href="https://github.com/dongjunchung/GPA_binary/blob/master/GPA_0.9-3.tar.gz?raw=true">https://github.com/dongjunchung/GPA_binary/blob/master/GPA_0.9-3.tar.gz?raw=true</a></p>

<p>Windows binary: <a href="https://github.com/dongjunchung/GPA_binary/blob/master/GPA_0.9-3.zip?raw=true">https://github.com/dongjunchung/GPA_binary/blob/master/GPA_0.9-3.zip?raw=true</a></p>
