    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-what-is-the-dspr" class="anchor" href="#what-is-the-dspr" aria-hidden="true"><span class="octicon octicon-link"></span></a>What is the DSPR?</h1>

<p>The <em>Drosophila</em> Synthetic Population Resource (DSPR) consists of a new panel of over 1700 recombinant inbred lines (RILs) of <em>Drosophila melanogaster</em>, derived from two highly recombined synthetic populations, each created by intercrossing a different set of 8 inbred founder lines (with one founder line common to both populations). Complete genome sequence data for the founder lines are available, and in addition, there is a high resolution genetic map for each RIL. The DSPR has been developed as a community resource for high-resolution QTL mapping and is intended to be used widely by the <em>Drosophila</em> community.</p>

<h1>
<a id="user-content-advantages-of-the-dspr" class="anchor" href="#advantages-of-the-dspr" aria-hidden="true"><span class="octicon octicon-link"></span></a>Advantages of the DSPR</h1>

<h2>
<a id="user-content-benefits-of-rils" class="anchor" href="#benefits-of-rils" aria-hidden="true"><span class="octicon octicon-link"></span></a>Benefits of RILs</h2>

<p>Because the DSPR consists of recombinant inbred lines, genotyping need only be carried out once, phenotypes that cannot easily be scored on single individuals can be assayed (i.e., those of low heritability and/or with appreciable measurement error), and RILs facilitate the examination of genotype-by-sex, genotype-by-environment, and so on, interactions.</p>

<h2>
<a id="user-content-wide-range-of-natural-variation" class="anchor" href="#wide-range-of-natural-variation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Wide range of natural variation</h2>

<p>The DSPR was created from a worldwide sample of 15 founder lines (South Africa, 2 lines; Asia, 4 lines; Australasia, 1 line; Europe, 2 lines; North America, 4 lines; South America, 2 lines). This panel therefore captures a large proportion of natural trait variation and provides a robust picture of trait genetic architecture.</p>

<h2>
<a id="user-content-extremely-high-mapping-resolution" class="anchor" href="#extremely-high-mapping-resolution" aria-hidden="true"><span class="octicon octicon-link"></span></a>Extremely high mapping resolution</h2>

<p>The DSPR is derived from highly recombined synthetic populations. This design allows for QTLs to be mapped to very narrow regions of the genome (1-2cM), encompassing just a handful of genes.</p>

<h2>
<a id="user-content-joint-estimate-of-qtl-effect-and-frequency" class="anchor" href="#joint-estimate-of-qtl-effect-and-frequency" aria-hidden="true"><span class="octicon octicon-link"></span></a>Joint estimate of QTL effect and frequency</h2>

<p>Each set of RILs is derived from eight inbred founders. This multi-founder composition permits an estimate of BOTH the effect and frequency of mapped QTL, allowing for an estimate of the contribution of each QTL to natural trait variation.</p>

<h2>
<a id="user-content-identification-of-causative-sites" class="anchor" href="#identification-of-causative-sites" aria-hidden="true"><span class="octicon octicon-link"></span></a>Identification of causative sites</h2>

<p>Researchers using the DSPR will be able to identify putative causative sites. For each mapped QTL, a QTL allele can be assigned to each founder. The whole genome re-sequencing data allows the small proportion of polymorphisms in phase with the QTL allelic configuration to be identified. This small set is highly likely to include the causative site.</p>

<a id="user-content-installing-the-dsprqtl-package" class="anchor" href="#installing-the-dsprqtl-package" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installing the DSPRqtl Package</h1>

<h2>
<a id="user-content-stable-release" class="anchor" href="#stable-release" aria-hidden="true"><span class="octicon octicon-link"></span></a>Stable Release</h2>

<h2>
<a id="user-content-development-version" class="anchor" href="#development-version" aria-hidden="true"><span class="octicon octicon-link"></span></a>Development Version</h2>

<code>install_github()</code> from the <a href="https://github.com/hadley/devtools">devtools
package</a>.</p>
