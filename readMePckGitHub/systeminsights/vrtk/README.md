    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-vrtk" class="anchor" href="#vrtk" aria-hidden="true"><span class="octicon octicon-link"></span></a>vrtk</h1>

<p>The vrtk (which stands for 'vimana research toolkit' or 'vimana R toolkit') is a set of tools to facilitate reading and working with data from the vimana software platform.</p>

<p>It is built to work with manufacturing data obtained from System Insights. </p>

<p>The motivation behind the project is simple -- to facilitate research using data obtained using the vimana platform ('vimana data', if you will). Anyone, anywhere, in the possession of <em>vimana data</em> should not have to reinvent the wheel in trying to import it into R in the right formats, converting timestamps, etc. This is where the vrtk comes into the picture.</p>

<p>The project is intended to be open-source, and the source-code is hosted here. The packages themselves can be downloaded using the links provided below.</p>

<ul class="task-list">
<li>The <a href="https://docs.google.com/file/d/0Bxr2Lj6RVx3JOFROMEdJVElNd0E/edit?usp=sharing">tar.gz</a> - Linux users</li>
<li>The <a href="https://docs.google.com/file/d/0Bxr2Lj6RVx3JQnQxNXo2ajlKUEU/edit?usp=sharing">zip</a> - Windows users</li>
</ul>

<p>There is also an example analysis, which you can download from this repository, in the folder 'example'.</p>

<h4>
<a id="user-content-contents" class="anchor" href="#contents" aria-hidden="true"><span class="octicon octicon-link"></span></a>Contents:</h4>

<p>The repo currently contains:</p>

<ol class="task-list">
<li>The source code for the R library, with documentation included.</li>
<li>A simple example analysis, in the example folder.</li>
</ol>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>The package can be installed directly from within R (in both Windows and Linux):</p>

<ol class="task-list">
<li>If you are a windows user, use the <a href="https://docs.google.com/file/d/0Bxr2Lj6RVx3JQnQxNXo2ajlKUEU/edit?usp=sharing">windows link</a>, and if you use linux, use <a href="https://docs.google.com/file/d/0Bxr2Lj6RVx3JOFROMEdJVElNd0E/edit?usp=sharing">the tar.gz</a>.</li>
<li>From the R command prompt, use the install.packages() function, and use the flag dependencies=TRUE if required.</li>
<li>The package should now be installed. If you encounter any problems with the documentation, restart R once.</li>
</ol>

<p>Caveat: If you face a 'paste0 not found' error, this is most likely due to an older R version. The package should work fine for R versions 2.15.2 and above.</p>

<h2>
<a id="user-content-compiling-from-source" class="anchor" href="#compiling-from-source" aria-hidden="true"><span class="octicon octicon-link"></span></a>Compiling from Source</h2>

<p>The vrtk is intended to be an open-source project.</p>

<p>If you wish to contribute to the vrtk project, or if the direct installation doesn't work for you, or even if you simply want to play with the code, you are welcome to download this repository (or even better, fork us!).</p>

<p>If you are new to compiling R packages from source (especially in windows!), you'll find that it can be tricky. We found the following guides and tools helpful. Make sure you take a look at these first:</p>

<ul class="task-list">
<li>The brilliant <a href="http://cran.r-project.org/web/packages/devtools/index.html">devtools</a> package, which can also be directly installed from CRAN. This has an <code>install_github</code> function that can install from repos like this one easily.</li>
<li>The official CRAN <a href="http://cran.r-project.org/doc/manuals/R-exts.html">guide to writing R extensions</a>.</li>
<li>The CRAN <a href="http://cran.r-project.org/bin/windows/base/rw-FAQ.html#Can-I-install-packages-into-libraries-in-this-version_003f">guide to compiling packages on windows</a>.</li>
<li>The <a href="http://cran.us.r-project.org/bin/windows/Rtools/">Rtools</a> for windows users.</li>
<li>Lastly, the <a href="http://win-builder.r-project.org/">win-builder.R project</a>.</li>
</ul>
