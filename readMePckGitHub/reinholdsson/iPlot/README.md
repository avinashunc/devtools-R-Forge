    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-iplot" class="anchor" href="#iplot" aria-hidden="true"><span class="octicon octicon-link"></span></a>iPlot</h1>

<p>A visualization app using click events for static plots in <a href="https://github.com/rstudio/shiny">shiny</a> (since 0.7.0).</p>

<pre><code>library(iPlot)
iPlot(ggplot2::diamonds)
</code></pre>

<p>Example apps on glimmer:</p>

<ul class="task-list">
<li><a href="http://glimmer.rstudio.com/reinholdsson/iplot/">iPlot with ggplot2::diamonds</a></li>
<li><a href="http://glimmer.rstudio.com/reinholdsson/iplot-survey/">iPlot with MASS::survey</a></li>
</ul>
