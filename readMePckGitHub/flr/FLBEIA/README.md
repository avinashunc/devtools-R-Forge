<a id="user-content-flbeia" class="anchor" href="#flbeia" aria-hidden="true"><span class="octicon octicon-link"></span></a>FLBEIA</h1>

<p><strong>Bio-Economic Impact Assessment of Management strategies using FLR</strong></p>

<p>Version: 1.0</p>

<p><code>FLBEIA</code> provides a simulation toolbox which facilitates the development of bio-economic impact assessments of fisheries management strategies. It is built under a Management Strategy Evaluation framework using FLR. The simulation is divided in two <em>worlds</em>: the operating model (OM, i.e. the real world) and the management procedure model (MP, i.e. the perceived world). The OM is itself divided in 3 components: biological, fleets and environmental and economic covariates. The MP is also divided in 3 components: data, the 
perceived system and advice.</p>

<p>Author: Dorleta GARCIA <a href="mailto:dgarcia@azti.es">dgarcia@azti.es</a></p>

<p>Maintainer: Dorleta GARCIA, AZTI Tecnalia.</p>
