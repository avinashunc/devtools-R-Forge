    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-choosecolor" class="anchor" href="#choosecolor" aria-hidden="true"><span class="octicon octicon-link"></span></a>choosecolor</h1>

<p>Quickly choosing colors and making palettes in R can be a chore.
The functions in <code>choosecolor</code> are a simple set of functions 
which can help to rapidly select colors in R.  Using the "locator"
functionality in R, it enables an interactive way of selecting colors 
visually for use in R graphics. It is built to be lightweight and
depends on nothing but the base R installation. </p>

<p>It's nothing fancy, just functional. </p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>Currently there isn't a release on <a href="http://cran.r-project.org/">CRAN</a>,
though there may one day be one. You can still<br>
download the <a href="https://github.com/MarcoDVisser/choosecolor/zipball/master">zip</a> 
or <a href="https://github.com/MarcoDVisser/choosecolor/tarball/master">tar ball</a>.
Then decompress and run <code>R CMD INSTALL</code> on it, 
or use the <strong>devtools</strong> package to install the development version
(following code courtesy of <a href="https://github.com/Dasonk">dasonk</a> ).</p>

<div class="highlight highlight-r"><pre><span class="pl-c">## Make sure your current packages are up to date</span>
update.packages()
<span class="pl-c">## devtools is required</span>
library(<span class="pl-smi">devtools</span>)
install_github(<span class="pl-s"><span class="pl-pds">"</span>choosecolor<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>MarcoDVisser<span class="pl-pds">"</span></span>)</pre></div>

<h2>
<a id="user-content-usage-examples" class="anchor" href="#usage-examples" aria-hidden="true"><span class="octicon octicon-link"></span></a>Usage Examples</h2>

<div class="highlight highlight-r"><pre><span class="pl-c"># pick a color</span>
<span class="pl-smi">mycol</span><span class="pl-k">&lt;-</span>color.choose()

<span class="pl-c">#make a pallete </span>
<span class="pl-smi">Mypalette</span><span class="pl-k">&lt;-</span>palette.picker(<span class="pl-v">n</span><span class="pl-k">=</span><span class="pl-c1">5</span>) 

<span class="pl-c"># use your palette e.g.</span>
Mypalette(<span class="pl-c1">10</span>)
