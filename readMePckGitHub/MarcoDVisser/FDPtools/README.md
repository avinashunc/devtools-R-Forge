    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a id="user-content-fdptools" class="anchor" href="#fdptools" aria-hidden="true"><span class="octicon octicon-link"></span></a>FDPtools</h1>

<p>Optimized tools for working with forest dynamics plot data. The package is an 
attempt to create many optimized analytical tools for calculating statistics
of interest from forest dynamics plot data. Tools range from spatail tools such
as distance calculation to Bayesian variable selection and inverse modelling.
The package is currently under development and contains source code in C 
that has only been tested under linux. Windows users should take care.</p>

<h2>
<a id="user-content-installation" class="anchor" href="#installation" aria-hidden="true"><span class="octicon octicon-link"></span></a>Installation</h2>

<p>You can download the <a href="https://github.com/MarcoDVisser/FDPtools">development version of FDPtools</a> as <a href="https://github.com/MarcoDVisser/FDPtools/zipball/master">zip</a> or <a href="https://github.com/MarcoDVisser/FDPtools/tarball/master">tar ball</a>. Then <code>R CMD INSTALL</code> on it, or use the <strong>devtools</strong> package to install the development version:</p>

<div class="highlight highlight-r"><pre><span class="pl-c">## Make sure your current packages are up to date</span>
update.packages()
<span class="pl-c">## devtools is required</span>
library(<span class="pl-smi">devtools</span>)
install_github(<span class="pl-s"><span class="pl-pds">"</span>FDPtools<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>MarcoDVisser<span class="pl-pds">"</span></span>)</pre></div>

<p>Microsoft Windows users will need to install <a href="http://www.murdoch-sutherland.com/Rtools/">Rtools</a> first.</p>

<h2>
<a id="user-content-examples" class="anchor" href="#examples" aria-hidden="true"><span class="octicon octicon-link"></span></a>Examples</h2>

<div class="highlight highlight-r"><pre><span class="pl-c"># Make spatial data</span>
require(<span class="pl-smi">FDPtools</span>)
<span class="pl-smi">x1</span><span class="pl-k">&lt;-</span>runif(<span class="pl-c1">10000</span>)
