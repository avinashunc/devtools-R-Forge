    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><p><a href="/rdinnager/greenpipeR/blob/master/A_green_pipe.gif" target="_blank"><img src="/rdinnager/greenpipeR/raw/master/A_green_pipe.gif" alt="a green pipe" style="max-width:100%;"></a></p>

<p><strong>version 0.3</strong></p>

<p>This package introduces the new pipe operator <code>%W&gt;%</code> — also known as the "warp" operator — which is inspired by a certain famous <em>plumber</em>. It does exactly the same thing as the <code>%&gt;%</code> operator from <a href="https://github.com/smbache/magrittr"><code>magrittr</code></a>: it pipes the left-hand side into an expression on the right-hand side, with one very small addition. When you use <code>%W&gt;%</code> it recreates what goes through my head (and probably yours if you are from a certain generation) everytime I use a pipe in R. So if you are a <code>magrittr</code> <em>plumber</em>, and you can't get enough of piping/warping objects into functions/dungeon levels, then you may enjoy this package.</p>

<p>It requires <a href="https://github.com/smbache/magrittr"><code>magrittr</code></a> (the latest version from github) and <a href="https://github.com/rasmusab/beepr"><code>beepr</code></a> (now on <a href="http://cran.r-project.org/web/packages/beepr/index.html">CRAN!</a>). You can install <code>greenpipeR</code> by running the following in R:</p>

<pre><code>## install magrittr dependency
devtools::install_github("smbache/magrittr")
## install greenpipeR
devtools::install_github("rdinnager/greenpipeR")
</code></pre>

<p><strong>Warning::</strong> If you have easily annoyed office mates, then use this package with caution. Also, in case you were wondering, this package is not <em>Useful</em>, only fun. Not suitable for those with high maturity level. Personally I now always use <code>%W&gt;%</code> in place of <code>%&gt;%</code>, but your mileage may vary.</p>

<p>If you haven't figured out by now what <code>%W&gt;%</code> does, make sure the sound on your computer is on (but not too loud!) before you use it. :)</p>
